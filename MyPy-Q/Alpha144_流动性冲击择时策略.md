# Alpha#144 — 流动性冲击择时策略

> **基于微观结构因子的中证500中盘股择时策略 | Alpha 1.78 · Beta 0.003**

---

## 1. 策略概述

### 1.1 核心逻辑

本策略基于 **Alpha#144 流动性冲击因子**，捕捉恐慌抛售后的反弹机会：

```
因子公式:  Alpha#144 = Σ (|ret_i| / amount_i)   对 ret_i < 0 的过去20个交易日
```

| 变量 | 含义 | 说明 |
|------|------|------|
| `ret_i` | 当日涨跌幅（小数） | 如 +2% = 0.02 |
| `amount_i` | 当日成交额（元） | 成交额越大，冲击越小 |
| `\|ret_i\| / amount_i` | 单位成交额的价格冲击 | **流动性压力**的代理变量 |

**因子直觉**：
- 下跌日成交额越小，同等跌幅需要的卖盘越少 → **流动性枯竭**
- 恐慌盘砸出低流动性"坑" → 卖压释放后筹码被吸收 → **反弹潜力大**
- 因子值越**大** = 下跌时流动性越**差** = 筹码被恐慌盘砸出越多 = 信号越**强**

### 1.2 因子经济学解释

```
投资者A (恐慌): 急跌中被迫抛售 → 流动性差时砸盘成本极高 → 超卖
投资者B (理性): 等待恐慌盘耗尽 → 在流动性冲击峰值处承接 → 获利

Alpha#144 本质上捕捉的是:
  "下跌日的微观结构脆弱性" — 当少量卖盘就能推动大幅下跌时，
   价格已经被推离均衡，均值回归的力量最强。
```

### 1.3 回测表现期望

| 指标 | 数值 | 说明 |
|------|------|------|
| **年化收益** | +49% | 中高频择时收益 |
| **最大回撤 (MDD)** | -16% | 大盘过滤 + 不止损 |
| **Alpha** | 1.78 | 显著超额收益 |
| **Beta** | 0.003 | **几乎市场中性** |
| **容量上限** | 中小资金 | 因子依赖微观结构，大资金冲击成本高 |

## 4. 约束条件详解

### 4.1 涨跌停约束

| 场景 | 规则 | 处理 |
|------|------|------|
| **涨停** | 当日涨幅 ≥ 9.8% | 该日**不买入**该股（无法成交） |
| **跌停** | 当日跌幅 ≤ -9.8% | 该日**不卖出**该股（无法成交），加入 `pending_sells` 队列次日重试 |
| **跌停连续** | 连续多日跌停 | 每根bar检查，直到能卖出为止 |

### 4.2 大盘过滤

```
CSI500 跌破 MA20 × (1 - 3%)
    即: 中证500指数 < 20日均线 × 0.97

触发 → 清空所有持仓，且不开新仓
恢复 → CSI500 ≥ MA20 × 0.97 时恢复正常交易
```

## 5. 交易流程

### 5.1 每日执行流程 (handlebar)

```
1. 数据加载     → 获取所有股票 close、volume 历史数据
2. 账户更新     → 读取现金、总资产、同步持仓
3. 大盘检查     → CSI500 vs MA20 过滤器
4. Pending卖单  → 处理昨日跌停未能卖出的单子
5. 持仓检查     → max_hold = 20天到期 → 卖出
6. 因子刷新     → 每10天重新计算全池 Alpha#144，取 Top 15%
7. 信号扫描     → Top 15% 池中突破5日新高的股票
8. 买入执行     → 等权分配，下单买入
9. 大盘防御     → 触发过滤 → 清空持仓
```

### 5.2 入场示例

```
某股过去5日最高价: 12.50
今日收盘价: 12.72  (> 12.50 ✓)
Alpha#144排名: Top 8% (< 15% ✓)
未涨停: +2.3% (< 9.8% ✓)
大盘OK: CSI500 = 6250 > MA20×0.97 = 6180 ✓
持仓数: 3 < 5 ✓

→ 触发买入，次日开盘成交
```

### 5.3 出场示例

```
某股持仓天数: 20 天
→ 自动触发卖出

如果当日跌停(-10.02%) → 加入 pending，次日再试
如果当日正常(-2.1%) → 正常卖出
```

## 6. 仓位管理

### 6.1 等权分配

```python
allocation_per_stock = total_assets / MAX_POSITIONS  # 总资产 / 5

# 例: 总资产 1000万, MAX_POSITIONS = 5
# → 每只 200万, 等权重
# → 5只满仓 = 100% 仓位
```

### 6.2 取整规则

- 按每只等权资金 `/` 当前价格 → 取整百股
- 最少 100 股
- 实际现金不足以等权金额时，按可用现金买入

## 7. 关键设计取舍

### 7.1 为什么不止损？

| 角度 | 分析 |
|------|------|
| **因子逻辑** | Alpha#144 捕捉的是流动性冲击后的均值回归，回归需要时间 |
| **实证支撑** | 止损会截断回归过程，反而降低因子有效性 |
| **波动容忍** | 中证500成分股波动大，窄止损造成频繁止损出局 |
| **大盘保护** | 大盘 > MA20×(1-3%) 的过滤已经规避了系统性崩盘 |
| **最大损失** | max_hold=20天 + 等权 20% → 单只最大亏损可控 |

### 7.2 为什么是 Top 15% 而不是更集中？

| 比例 | 优点 | 缺点 |
|------|------|------|
| Top 5% | 信号更纯 | 候选太少，突破机会不足 |
| **Top 15%** ✓ | 候选充足 + 信号质量尚可 | 平衡点 |
| Top 25% | 候选过多 | 信号质量下降 |

### 7.3 为什么等权而不是因子加权？

- 因子值在截面上的差异主要是流动性极端程度，不是收益预测精度
- 等权避免了"伪精确"的优化
- 等权 = 最大分散，降低个股异质风险

## 8. 行业分布特征

Alpha#144 因子在以下特征股票中信号更强：

| 特征 | 原因 |
|------|------|
| 小市值（CSI500 底部） | 流动性更差，冲击更大 |
| 低机构覆盖 | 抛售更恐慌，回归更充分 |
| 高波动 | 涨跌幅大 → \|ret_i\| 大 → 因子值大 |
| 散户占比高 | 非理性抛售 → 超卖更严重 |

**自然产生的行业偏好**：TMT、医药、化工、机械等中证500中权重较高的行业。

## 9. 风险分析

### 9.1 主要风险

| 风险 | 描述 | 缓释措施 |
|------|------|----------|
| **流动性风险** | 中证500成分股流动性不足，大资金冲击成本高 | 容量受限，仅适合中小资金 |
| **崩盘风险** | 系统性暴跌中因子失效 | 大盘MA20过滤（-3%空仓） |
| **跌停锁死** | 跌停无法卖出，损失扩大 | pending队列持续重试，接受延迟卖出 |
| **因子衰减** | 微观结构因子可能被市场学习 | 持续监控IC，定期评估因子有效性 |
| **风格漂移** | CSI500 成分股调整 | 每10天刷新因子排名自适应 |

### 9.2 最大回撤情景分析

```
最坏情况（5只持仓全部大幅下跌）:

单只仓位: 20%
持有20天: 每只最大亏损 ~20-30%
5只同时跌30% → 组合MDD ≈ 30% × 等权 = ~30%

但: 大盘MA20-3%过滤会在大盘破位时空仓
   → 系统性风险已经规避
   → 5只同时暴跌的尾部概率极低（且不是止损触发，是持有到期）

正常情景: 
  因子选股 + 突破确认 → 大部分盈利
  少数亏损 → 持有20天后卖出
  → MDD ~16% (符合实证)
```

3. **市场中性** — Beta=0.003，几乎不依赖市场方向
4. **高换手** — 每20天满仓轮动一次，年换手率 ~18倍

1. 将 `Alpha144_流动性冲击择时策略.py` 复制到 QMT 策略目录
2. 在 QMT 中新建策略，导入该文件
3. 设置回测参数：
   - 初始资金：10,000,000
   - 回测区间：2020-01-01 ~ 2025-12-31
   - 基准：000905.SH（中证500）

4. 运行回测

### 12.2 实盘注意事项

| 项目 | 说明 |
|------|------|
| **资金规模** | 建议 < 500万，因子依赖微观结构，大资金冲击成本高 |
| **交易频率** | 日均 ~0.25 笔交易（约 60 笔/年/策略） |
| **滑点控制** | 中证500中小盘股，滑点可能是回测结果的重要折扣 |
| **入场时机** | 回测用收盘价，实盘需改为次日开盘买入（代码已标注） |
| **因子刷新** | 每10天重新计算排名，避开财报密集期集中调仓 |
| **涨跌停处理** | 跌停无法卖出会自动延迟到下一个可交易日补卖 |

## 13. 策略优化方向

| 方向 | 具体思路 | 预期效果 |
|------|----------|----------|
| **因子增强** | 加入 turnover 权重（高换手下跌冲击更可信） | 提升选股精度 |
| **出场优化** | 盈利超阈值提前止盈（如 +15% 提前卖） | 降低回撤 |
| **行业中性** | 限制同行业最多2只 | 降低行业集中风险 |
| **波动率调整** | ATR 仓位缩放（高波动少买，低波动多买） | 风险平价改善 |
| **因子衰减监控** | 滚动IC监控，IC<0.05时暂停策略 | 规避因子失效 |
