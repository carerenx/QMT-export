## 七、核心洞察

### 7.1 评分相关性始终为零

```
三个回测周期（2024→2025, 2025→2026）一致性发现:
  - 评分与未来收益的相关性始终≈0
  - 高评分 ≠ 高收益
  - 低评分 ≠ 低收益（反而经常是超级牛股）

技术面评分能回答的问题:
  ✅ "这只股票趋势好不好？"（趋势确认）
  ❌ "这只股票能涨多少？"（涨幅预测）
```

### 7.2 行业过滤是唯一有效的优化

| 优化手段 | 效果 |
|----------|------|
| 加权评分(V2) | ❌ 无效，相关性恶化 |
| 新增底部蓄力(V2) | ❌ 仅捕获4只新标的 |
| **行业过滤(V3)** | ✅ **胜率从52%→65~91%** |
| 评分调参 | ❌ 相关性始终≈0 |

### 7.3 策略的正确使用方式

```
正确: 科技行业 + 技术面趋势确认 = 大概率参与牛市（胜率65-91%）
错误: 技术面评分排名选股 = 与随机无异（相关性≈0）

策略价值 = 确保你在正确的赛道上 + 趋势已经启动
超额收益 = 来自行业β + 个股基本面α（非技术面）
```
