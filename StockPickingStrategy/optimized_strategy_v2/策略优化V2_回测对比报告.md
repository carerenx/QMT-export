## 四、V2 的改进与退步

### 4.1 ✅ 改进之处

| 改进 | 证据 |
|------|------|
| **排除弱势行业** | 白酒(600809 V2=98但V2标记为"无行业优势")、房地产相关标的被压低评分 |
| **底部蓄力捕获** | 002080(玻璃玻纤) V1=47分→V2=62分, 实际+167% |
| **多时间框架** | 周线下跌的标的被扣分，减少了部分假突破 |

### 4.2 ❌ 退步或无效之处

| 问题 | 证据 |
|------|------|
| **相关性恶化** | V1=0.027 → V2=-0.022（略微负相关！） |
| **高分池扩大但质量下降** | 47只vs24只，均收益从87%降到63% |
| **假阳性未改善** | V2=105分的000727实际-0.4%，600776仅+12.8% |
| **长飞光纤仍未捕获** | 601869 V2可能仍然低分（V1仅39分） |

### 4.3 根本问题：为什么评分无法预测收益排名？

```
技术面评分的本质 = 识别"趋势已经启动"的股票
未来收益的决定因素 = 行业景气度 × 个股基本面 × 资金面 × 市场情绪

技术面只能回答: "趋势在不在?"
不能回答: "趋势能走多远?"

这就是为什么:
  - 600105(光纤, V2=105)涨了559%  ← 行业景气+基本面强
  - 600809(白酒, V2=98)亏了48%     ← 行业下行（白酒周期）
  - 000727(面板, V2=105)几乎持平   ← 行业中性+个股一般
  
三只股票在2025年6月技术面都"完美"，但行业β差异导致结局天壤之别。
```

## 五、V3.0 优化方向（基于V2回测教训）

### 5.1 必须加入的维度

| 优先级 | 维度 | 可行性 | 预期效果 |
|--------|------|--------|----------|
| 🔴 P0 | **行业营收增速** | 财报API可获取 | 高（直接过滤下行行业） |
| 🔴 P0 | **北向资金/主力资金** | 东财资金流API | 高（排除资金出逃的伪突破） |
| 🟡 P1 | **机构覆盖变化** | 同花顺一致预期 | 中（排除无机构关注的边缘股） |
| 🟡 P1 | **板块联动确认** | 同板块≥3只共振 | 中（排除孤立的个股行情） |
| 🟢 P2 | **股东户数变化** | 东财datacenter | 低（季度数据，滞后） |

### 5.2 评分模型重构建议

```
当前问题: 纯技术面评分 = 噪声放大器
           → 高分不能区分+500%和-50%

V3方案: 分层过滤代替加权评分

  第一层（技术面）: 40分阈值 → 淘汰弱股（保留60-70%候选）
  第二层（行业面）: 行业景气度 → 淘汰下行行业（保留40-50%）
  第三层（资金面）: 主力资金方向 → 淘汰出货标的（保留20-30%）
  第四层（基本面）: 营收增速+ROE → 精选最终标的（保留10-15%）
  
  最终输出: 8-15只精选标的，而非排名列表
```

## 六、V1 vs V2 最终结论

### 6.1 数值对比

```
                    V1.0        V2.0        变化
评分-收益相关性      0.027      -0.022       ↓ 无改善
高分池均收益        +86.7%     +63.0%       ↓（池子更大）
高分池胜率           54%        53%         → 无变化
捕获超级牛股(>200%)  4只        6只          ↑ +2只
新捕获标的           —         4只(V1漏)    🆕 含+167%
```

### 6.2 诚实的结论

```
V2.0相比V1.0有边际改善（多捕获2只超级牛股），但没有质的飞跃。

根本原因:
  技术面评分的天花板很低——它能回答"趋势好不好"，
  但不能回答"趋势能持续多久、能走多远"。
  
  后者由行业景气度、个股基本面、资金面共同决定，
  而这些是纯技术面评分无法量化的。

V3.0的方向应该是:
  放弃"用技术面预测涨幅排名"的幻想，
  转向"技术面初筛 + 行业面过滤 + 资金面确认 + 基本面精选"
  的分层决策体系。
```

### 6.3 实战建议

```
当前(2026年6月)使用V2.0策略:
  1. 运行筛选，关注评分≥75的标的（约30只）
  2. 手动排除: 白酒/地产/保险/钢铁/煤炭等下行行业
  3. 优先关注: 通信设备/半导体/玻璃玻纤/电子化学品行业
  4. 最终决策前: 检查Q1财报（营收增速>15%, ROE>10%）
  5. 仓位: 等权5-8只，单只<20%，总仓位60-80%
```
