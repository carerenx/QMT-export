# "前期发现"选股策略 V1.0

> 目标：在全市场主板中，通过技术面多维评分，发现处于早期启动阶段的潜力标的

## 文件结构

```
strategy_v1_initial/
├── README.md                              ← 本文件
├── 选股策略_前期发现_V1.0.md               ← 策略完整文档（评分模型/数据采集/过滤规则）
├── 选股策略回测_2025年6月至2026年6月.md     ← 历史回测报告（110只标的，12个月跟踪）
├── screen_similar_stocks.py               ← 实时筛选脚本（可每日运行）
├── backtest_2025_strategy.py              ← 回测脚本（可复现）
├── data_screening/                        ← 2026-06-28实时筛选结果
│   ├── candidates_scored.csv               ← 125只候选标的的完整评分
│   └── kline_*.csv (20 files)              ← TOP20标的的60日K线
└── data_backtest/                          ← 回测结果数据
    └── backtest_results.csv                ← 110只标的的评分+实际收益
```

## 核心发现

| 指标 | V1.0表现 |
|------|----------|
| 评分-收益相关性 | 0.027（几乎为零） |
| 高分(≥80)均收益 | +86.7% |
| 高分胜率 | 54% |
| 最大漏网之鱼 | 长飞光纤(评分39, 实际+664%) |
| 假阳性率 | 46%（高分但亏损） |

## 策略局限

- 纯技术面评分无法区分真假突破
- 无法预测涨幅排名
- 可能错过底部反转机会
- 需要行业/基本面辅助过滤
