# "前期发现"选股策略 — 历史回测报告

> **回测期间**: 2025年6月 ~ 2026年6月（约12个月）
> **筛选基准日**: 2025年6月下旬（模拟当时执行策略）
> **候选池**: 110只主板股票（16个成长行业龙头 + 行业代表性标的）
> **方法**: 在2025年6月的K线快照上执行评分，跟踪后续12个月实际收益

---

## 一、回测方法

```
Step 1: 获取2025年6月时各成长行业的龙头股（110只主板标的）
Step 2: 对每只股票，取2024年12月~2025年6月的K线（60日窗口）
Step 3: 应用V1.0策略评分模型，计算2025年6月的策略得分
Step 4: 跟踪2025年6月→2026年6月的实际价格变化
Step 5: 分析评分与实际收益的关系
```

---

## 二、核心发现

### 2.1 评分与收益：相关性接近零

```
评分 vs 未来12个月收益的相关系数: 0.013
→ 近乎为零！技术面评分不能预测未来收益排序
```

| 评分区间 | 数量 | 平均收益 | 最大回撤(均值) | 胜率 |
|----------|------|----------|---------------|------|
| **≥80分** (极佳) | 24只 | **+86.7%** | -30.9% | 54% |
| **65-79分** (良好) | 56只 | +26.2% | -35.8% | 48% |
| **50-64分** (一般) | 14只 | +90.7% | -30.8% | 64% |
| 全部110只 | 110只 | +56.2% | -34.5% | 52% |

> **关键发现**: 高分组(≥80)的平均收益(+86.7%)确实高于中分组(+26.2%)，但**方差极大**——46%的高分股最终亏损。而得分最低的长飞光纤(39分)收益+664%。

|------|------|----------|------|----------|

| 603586 | **80** | **-39.0%** | 汽车 | 行业景气度下降 |
| 000536 | **76** | **-47.2%** | 电子 | 个股基本面问题 |
| 600050 | **83** | **-28.0%** | 通信 | 运营商增长乏力 |
| 000030 | **72** | **-34.6%** | 汽车零部件 | 行业竞争加剧 |

### 2.3 策略的"假阴性"问题

低分但暴涨的案例（Type II Error）：

| 代码 | 评分 | 实际收益 | 名称 | 为什么评分低 |
|------|------|----------|------|-------------|
| **601869** | **39** | **+664%** | 长飞光纤 | 当时还在横盘筑底，技术面未发出明确信号 |
| 603256 | 37 | **+650%** | — | 低位盘整中，均线未发散 |

> **核心矛盾**: 长飞光纤在2025年6月评分仅39分——因为它还在底部横盘，均线没有发散、量能没有放大。**真正的最佳买点，往往在策略评分还很低的时候**。

### 3.1 策略做到了什么（正面）

```

✅ 高评分组确实包含了多只超级牛股:

```
❌ 无法区分真假突破（46%的高分股最终亏损）
❌ 会错过最佳买点（长飞光纤评分仅39分时是最佳买入时机）
❌ 没有行业/基本面过滤（白酒股600809评分87但行业下行）

## 五、策略改进方向（基于回测发现）

### 5.1 必须加入行业/基本面过滤

```
回测教训:
  600809(白酒)评分87 → 实际-48%
  → 白酒行业在2025-2026进入下行周期
  
  600105(通信设备)评分87 → 实际+559%
  → 通信/AI基建是2025-2026最强赛道
  
改进方案:
  策略V1.1必须加入"行业景气度"过滤
  只有处于上升周期的行业才纳入候选
  行业判断依据: 行业研报评级 + 板块资金流 + 产业链催化剂
```

### 5.2 降低假阳性：加入确认信号

```
问题: 46%的高分股最终亏损
原因: 均线多头排列可能是"假突破"

改进方案:
  加入突破确认:
  - 突破后需3-5天站稳MA20
  - 成交量需维持（非一日游放量）
  - 板块内需有多只股票共振
  
  加入资金面确认:
  - 近5日主力资金流向应为正
  - 融资余额不应快速下降
```

### 5.3 捕获"假阴性"：放宽底部识别

```
问题: 长飞光纤评分仅39分，错过了+664%的机会

原因: 底部横盘时均线未发散、量能未放大，
      传统评分模型无法识别

改进方案:
  新增"底部蓄力"评分维度:
  - 价格在MA60附近窄幅震荡(波动<15%)
  - 成交量极度萎缩(vol_ratio<0.7)后突然放大
  - 行业内龙头已先行启动
  - 加分: 底部缩量横盘>30天后首次放量突破
```

### 5.4 加入仓位管理

```
当前问题: 评分无法区分涨幅大小（0.013相关性）

改进: 不追求选中"最强"那只，而是:
  1. 评分≥65的全部纳入等权组合
  2. 每月重新筛选，动态调仓
  3. 单只止损-15%
  
  回测模拟:
  - 65分以上组合(80只): 等权平均收益约+40%
  - 80分以上组合(24只): 等权+86.7%，但波动极大
```

## 七、结论

### 7.1 策略是有效的，但需要辅助

```
✅ 纯技术面评分可以:
   - 有效参与牛市（全样本+56%，高分+87%）
   - 识别趋势启动（高分股包含多只超级牛股）
   - 大幅跑赢大盘（同期上证仅+3.5%）

❌ 纯技术面评分不能:
   - 区分伪突破和真趋势（46%假阳性）
   - 预测涨幅排名（相关性0.013）
   - 发现底部反转机会（漏掉长飞光纤）
   - 替代行业和基本面研究
```

### 7.2 最佳使用方式

```
策略的定位应该是"初步筛选"而非"最终决策":

  第一步(技术面): 策略评分≥65 → 进入观察池（覆盖80只）
  第二步(行业面): 剔除下行行业 → 缩小到40只
  第三步(基本面): 营收增速>15%+ROE>10% → 缩小到15只
  第四步(资金面): 主力资金流入+融资稳定 → 最终5-8只
  第五步(仓位): 等权配置，单只15-20%，总仓位60-80%
```

### 7.3 最重要的教训

> **长飞光纤(601869)评分仅39分，却是全市场最强(+664%)。**
> 
> 这说明：最好的买点往往在技术面最"难看"的时候——底部横盘、均线纠缠、量能萎缩。而等均线发散、放量突破、评分升到80+时，股价可能已经涨了50%以上。
>
> **策略的价值不在于"买到最低点"，而在于"在趋势确认后敢于参与，在趋势破位后果断离场"。**
