# V3.0：科技赛道专属策略 — 回测报告

> **核心优化**: 剔除历史上导致亏损的非科技行业，仅保留8个科技相关板块
> **回测期间**: 2025年6月 → 2026年6月（12个月）
> **对比基准**: V1.0全行业策略（110只标的）

---

## 一、优化逻辑：为什么要做行业过滤？

### 1.1 V1/V2的核心教训

```
V1.0全行业回测中最惨的高分假阳性:
  600809(白酒):   评分87 → 实际-48.0%  ← 白酒行业下行周期
  002459(光伏):   评分77 → 实际-37.0%  ← 光伏产能过剩
  000030(汽车零部件): 评分83 → -34.6%  ← 汽车价格战
  603586(汽车):   评分80 → -39.0%  ← 产业链转移

这些股票在2025年6月技术面完美（多头排列+均线发散+温和放量），
但因为所属行业处于下行周期，技术面的强势无法转化为持续收益。
```

### 1.2 V3的解决方案

```
剔除: 白酒, 银行, 保险, 房地产, 钢铁, 煤炭, 食品饮料, 家电,
      光伏设备, 风电设备, 电力设备, 汽车零部件, 汽车, 
      通用设备, 专用设备, 工程机械, 电池

保留: 8个科技相关板块
  ├── 通信设备    (光模块/光纤/5G/AI算力)
  ├── 半导体      (芯片/设备/材料)
  ├── 光学光电子  (面板/LED/光电子)
  ├── 电子化学品  (光刻胶/电子气体)
  ├── 元件        (电容/电阻/PCB)
  ├── 计算机设备  (服务器/AI算力硬件)
  ├── 国防军工    (军工电子/航天)
  └── 玻璃玻纤    (电子玻璃/光纤材料)
```

## 三、各科技子板块表现

| 板块 | 数量 | 平均收益 | 胜率 | 评价 |
|------|------|----------|------|------|
| **半导体** | 1只 | **+475.5%** | 100% | 🏆 最强（仅603061） |
| **元件** | 12只 | **+148.7%** | **100%** | 🏆 全胜！无一亏损 |
| **通信设备** | 12只 | **+131.6%** | 67% | 🥇 超级牛股集中营 |
| **玻璃玻纤** | 12只 | **+124.5%** | 50% | 🥈 高波动，但有603256(+650%) |
| **电子化学品** | 3只 | +45.1% | 100% | 🥉 稳定正收益 |
| 计算机设备 | 8只 | +23.4% | 38% | ⚠️ 分化严重 |
| 国防军工 | 5只 | +11.7% | 60% | ⚠️ 收益偏低 |
| **光学光电子** | 10只 | **-2.2%** | 30% | 🔴 唯一负收益科技板块 |

### 板块详解

```
元件(12只, +149%, 100%胜率): 
  → 电容/电阻/PCB是AI算力的基础元器件，需求确定性最强
  → 代表: 000636(+403%), 002484(+302%), 600183(+278%)

通信设备(12只, +132%, 67%胜率):
  → 光模块/光纤是AI数据中心互联的核心，超级牛股最多
  → 代表: 600105(+559%), 600487(+472%), 002281(+277%)
  → 但也有002017(-54%)这样的掉队者

光学光电子(10只, -2.2%, 30%胜率):
  → 面板/LED行业竞争激烈，即使技术面好也难以持续
  → V4应考虑剔除或降权
```

## 六、V3 vs V1 vs V2 最终总结

| 版本 | 策略 | 标的数 | 平均收益 | 胜率 | 高分均收益 |
|------|------|--------|----------|------|-----------|
| V1.0 | 全行业+技术面 | 110 | +56.2% | 52% | +86.7% |
| V2.0 | V1+底部蓄力+行业动量+周线 | 110 | +56.2% | 52% | +63.0% |
| **V3.0** | **科技赛道+技术面** | **68** | **+97.6%** | **65%** | **+116.4%** |

```
最大改进: V1 → V3
  +41.4pp平均收益提升
  +13pp胜率提升
  行业过滤是最有效的单一优化手段

V2的失败教训:
  在评分模型里加维度不如在候选池里做减法
  排除差行业 >> 优化评分公式
```

## 七、V4展望

```
基于V3回测发现:
  1. 光学光电子应降权或剔除（-2.2%均收益）
  2. 元件和通信设备应加大权重（+149%/+132%）
  3. 板块内需加入基本面过滤（排除亏损/ST/市值过小）
  4. 加入资金流确认（5日主力净流入为正）
```
