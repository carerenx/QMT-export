# V6.0：分层仓位+T3排除 — 终极回测报告

> **核心优化**: T3板块排除 + 收紧阈值 + 三级分层仓位
> **回测覆盖**: 两个完整年度周期（2024→2025 + 2025→2026）
> **对比基准**: V3.0 / V5.0

## 一、V5 → V6 核心变化

| 维度 | V5.0 | V6.0 | 效果 |
|------|------|------|------|
| **T3板块** | 纳入但提高阈值 | **直接排除** | 消除最大拖累 |
| **底部阈值** | ≥13（T3≥15） | **≥15（仅T1+T2）** | 假阳性减半 |
| **趋势阈值** | ≥55 | **≥50** | 扩大趋势覆盖 |
| **仓位分配** | 等权 | **分层权重（60/30/10）** | 精度更高 |
| **候选池** | 85只 | **49只（T1+T2）** | 池子减半 |

### 为什么排除T3？

```
两个周期的证据:
  光学光电子: 2025→2026胜率30%，均收益-2.2%
  计算机设备: 2025→2026胜率38%
  国防军工:   2025→2026胜率60%，均收益+12%

  → T3板块在分化市中严重拖累组合
  → 即使在2024→2025牛市中，T3也未显著优于T1
  → 排除T3 = 牺牲机会换确定性
```

## 六、V6策略规范

### 6.1 候选池

```
板块限定: 仅T1+T2
  T1: 通信设备(BK0448), 半导体(BK1036), 元件(BK0459)
  T2: 玻璃玻纤(BK0546), 电子化学品(BK1039)
  
排除: T3(光学光电子/计算机设备/国防军工)
排除: 创业板(300/301), 科创板(688/689), 北交所(8xx)
排除: ST/*ST, PE<0, 价格<5, 日均成交<2000万
```

### 6.2 选股信号（三级）

```python
# 第一层 TREND（趋势确认）- 仓位60%
trend >= 50  # 多头排列 + 均线发散 + 温和放量 + 早期位置

# 第二层 CORE（双轨共振）- 仓位30%
trend >= 40 AND bottom >= 12  # 趋势+底部同时确认

# 第三层 BOTTOM（底部蓄力）- 仓位10%
bottom >= 15  # 波动收缩 + 振幅收窄 + 缩量

# 任一满足即入选，优先级: TREND > CORE > BOTTOM
```

### 6.3 仓位分配

```
单只标的: 总资金 / 入选数量 × 层级权重
  TREND标的: 基准仓位 × 1.5
  CORE标的:  基准仓位 × 1.0
  BOTTOM标的: 基准仓位 × 0.5

组合层面:
  入选≤10只: 单只10%，满仓
  入选10-20只: 单只5-7%，总仓位70-90%
  入选>20只: 单只3-5%，总仓位60-80%
```
