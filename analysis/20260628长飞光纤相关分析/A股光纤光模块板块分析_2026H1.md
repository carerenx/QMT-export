### 4.2 资金流向关键结论

```
板块资金全景（120日）:
├── 总主力净流出: -2,278亿
├── 近20日流出: -726亿 (加速)
├── 近5日流出: -557亿 (急剧加速)
│
├── 唯一净流入: 源杰科技 +26亿 ← 光芯片龙头
├── 近20日逆势流入: 剑桥科技 +14亿, 博创科技 +10亿
│
└── 最大流出: 中际旭创 -530亿 ← 光模块市值龙头被系统减持

资金运动方向: 大市值光模块 → 小市值光模块 + 光芯片
              龙头减持       游资博弈        机构加仓
```

> **核心信号**: 板块整体涨幅+159%，但120日主力净流出-2,278亿。这是典型的**高位换手**——早期主力在上涨过程中将筹码卖给追涨的散户和后来者。源杰科技(+26亿)是唯一的例外，说明机构对**光芯片**子赛道的前景最为看好。

