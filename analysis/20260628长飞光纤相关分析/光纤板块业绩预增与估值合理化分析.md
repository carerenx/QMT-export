## 二、PEG 框架：业绩需要增多少？

```
PEG = PE(TTM) ÷ (EPS年增长率% × 100)

PEG < 1.0   →  低估（便宜，值得买入）
PEG 1.0~1.5 →  合理（价格与增长匹配）
PEG 1.5~3.0 →  偏贵（透支了1-2年增长）
PEG > 3.0   →  泡沫（脱离基本面）
```

## 三、三种"使涨幅合理"的走势情景

### 情景A：业绩爆发式增长（最佳路径）

股价不动，靠EPS增长把PE消化下来。

```
亨通光电 (PE=85x, 价=111元):

  假设2026年EPS增长85%（非常乐观）:
    2026年末EPS = 原EPS × 1.85
    若股价仍111元 → PE = 85/1.85 = 46x ← PEG=0.54（低估！）
  
  现实检验:
    亨通2025年净利约32亿
    85%增长需要2026年净利达到59亿
    光纤价格涨50% + 海缆订单翻倍 → 有可能触及
    但需要完美执行，容错空间极小

中际旭创 (PE=94x, 价=1254元):

  假设EPS年增50%:
    第1年末: PE = 94/1.5 = 63x
    第2年末: PE = 63/1.5 = 42x
    第3年末: PE = 42/1.5 = 28x ← 合理
  
  结论: 需要3年50%增速，股价横盘不涨
        这在光模块行业历史上是可行的（2023-2025年已有先例）

长飞光纤 (PE=388x, 价=543元):

  假设EPS年增100%（极其乐观）:
    第1年末: PE = 388/2 = 194x
    第2年末: PE = 194/2 = 97x
    第3年末: PE = 97/2 = 49x
    第4年末: PE = 49/2 = 24x ← 需要4年！
  
  结论: 即使每年EPS翻倍，也需要4年消化
        而EPS连续4年翻倍在制造业中从未发生过
```

### 情景B：股价回调 + 业绩增长（最健康的方式）

双向压缩PE，最快回归合理。

```
中际旭创 (当前PE=94x, 价=1254元):

  目标PE = 50x（假设50%增长，PEG=1.0）
  
  路径1（纯业绩）: EPS需增长88% → 几乎不可能一年完成
  路径2（纯回调）: 股价需跌47%到665元 → 可能但痛苦
  路径3（组合★）: EPS增40% + 股价跌25%到940元 → PE=50x ✓
  
  最可能路径: 股价回调30%到880元 + EPS增40% → PE=48x

长飞光纤 (当前PE=388x, 价=543元):

  目标PE = 100x（假设100%增长，PEG=1.0）
  
  所需: EPS增100% + 股价跌74%到141元
       或 EPS增100% + 横盘4年（情景A）
  
  结论: 无论哪条路径，当前543元的股价都无法被业绩合理化
```

### 情景C：纯靠股价下跌（最残酷但最常见）

EPS不增长的情况下，完全靠跌价来消化估值。

```
如果所有标的下季度EPS与去年持平（零增长）:

  中天科技: 需跌51%到30元 → PE=33x
  新易盛:   需跌55%到255元 → PE=33x
  中际旭创: 需跌68%到401元 → PE=30x
  亨通光电: 需跌65%到39元 → PE=30x
  长飞光纤: 需跌92%到44元 → PE=30x  ← 从543跌到44
  源杰科技: 需跌95%到89元 → PE=30x  ← 从1777跌到89
```

## 四、公式速查

```python
import math

# 1. 当前PE需要多少EPS增速才合理？
def required_growth_for_peg(pe, peg_target=1.0):
    """PEG = PE / (growth% * 100) → growth% = PE / (peg * 100)"""
    return pe / (peg_target * 100)

# 2. PE消化到目标PE需要几年？
def digestion_years(current_pe, target_pe, cagr):
    """消化年数 = ln(PE/目标PE) / ln(1+CAGR)"""
    if current_pe <= target_pe or cagr <= 0:
        return 0
    return math.log(current_pe / target_pe) / math.log(1 + cagr)

# 3. 给定增速，合理PE是多少？
def fair_pe(growth_rate, peg=1.0):
    """合理PE = 增速% * PEG * 100"""
    return growth_rate * peg * 100

# 4. 给定增速，合理股价是多少？
def fair_price(growth_rate, current_eps, peg=1.0):
    """合理股价 = 合理PE × 当前EPS"""
    return fair_pe(growth_rate, peg) * current_eps

# 5. 股价需要跌多少才能让PE回到合理？
def required_correction(current_pe, fair_pe):
    """回调幅度 = 1 - 合理PE/当前PE"""
    return 1 - fair_pe / current_pe
```

### 实战示例

```python
# 例1: 亨通光电
pe = 85        # 当前PE
growth = 0.85  # 假设EPS年增85%
fair = fair_pe(growth, peg=1.0)  # → 85x (刚好匹配!)
correction = required_correction(pe, fair)  # → 0%（不需要回调）
# 结论: 如果EPS真的能增85%，当前PE可被业绩消化

# 例2: 长飞光纤
pe = 388       # 当前PE
growth = 1.00  # 假设EPS年增100%（极其乐观）
fair = fair_pe(growth, peg=1.0)  # → 100x
correction = required_correction(pe, fair)  # → 74%（仍需跌74%!）
# 结论: 即使EPS翻倍，股价仍需跌74%才合理

# 例3: 新易盛
pe = 74        # 当前PE
growth = 0.50  # 假设EPS年增50%
fair = fair_pe(growth, peg=1.0)  # → 50x
correction = required_correction(pe, fair)  # → 32%
years = digestion_years(pe, 30, 0.50)  # → 2.2年
# 结论: 回调32%或横盘2.2年，估值可消化
```

## 六、核心结论

### 6.1 涨幅合理化的三个必要条件

```
① 业绩增速 ≥ PE / 100
   PE=100x → 需EPS年增 ≥ 100%；PE=400x → 需EPS年增 ≥ 400%

② 增速可持续 ≥ PE消化所需年数
   消化年数 = ln(PE/30) / ln(1+增速)

③ 股价涨速 ≤ 业绩增速
   半年涨452%但EPS只增50% → PE膨胀8倍 → 不可持续
```

### 6.2 光纤板块终极判断

```
15只核心标的:
├── 2只（中天科技/新易盛）：估值可被业绩消化 ← 相对安全
├── 2只（中际旭创/亨通光电）：需要极强的业绩 ← 不确定性大
├── 7只（光迅/天孚/太辰光/博创/仕佳/菲菱/剑桥）：PE超130x ← 需大幅回调
└── 4只（长飞/源杰/德科立/铭普）：PE超388x或亏损 ← 纯泡沫

根本规律:
  当PE超过100x，所需EPS增速(>100%/年)在制造业中几乎不可能实现
  因此高PE股票最终都以【暴跌】而非【横盘】来消化估值
  因为业绩永远追不上被情绪驱动的股价

唯一例外:
  如果有颠覆性技术突破（如光芯片完全国产替代、CPO技术商用化）
  可以在短期内实现200%+的EPS暴增，来支撑当前估值
  但这个概率——非常小
```
