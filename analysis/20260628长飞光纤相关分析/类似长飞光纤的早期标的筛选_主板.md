## 四、"前期发现"的关键信号组合

长飞光纤在2025年中期给出以下**可识别的早期信号**，本次筛选正是复刻这套逻辑：

### 4.1 必看信号（权重最高）

```
✅ 价格刚站上MA60，且距MA60 < 25%
   → "早期"的最核心定义：趋势确立但未透支
   → 长飞在2025年6月：距MA60约15%

✅ 均线多头排列（MA5>MA10>MA20>MA60）
   → 短中长期资金达成共识
   → 长飞在2025年7月开始出现

✅ 成交量温和放大（1.2x ~ 2.5x均量）
   → 资金开始关注，但尚未过热
   → 长飞在2025年7月：量比1.5x左右
```

### 4.2 加分信号

```
🟢 近期创20日新高 → 突破确认
🟢 20日涨幅5-25% → 稳健上行
🟢 回撤<6% → 资金惜售，筹码锁定好
🟢 PE<50x → 有基本面支撑
🟢 板块龙头 → 行业地位带来的确定性
```

### 4.3 警惕信号（即使评分高也要小心）

```
🔴 ST/*ST股票 → 退市风险（本次已过滤）
🔴 距MA60>80% → 可能已透支未来涨幅
🔴 成交量比>4x → 散户蜂拥，短期见顶风险
🔴 PE<0或>300x → 基本面有问题
🔴 20日涨幅>60% → 短期过热
```

## 五、TOP 5 重点标的深度分析

### 5.1 宏柏新材(605366) — ⭐ 最接近长飞光纤早期阶段

```
评分: 89分
赛道: 光纤材料（含硫硅烷+高纯四氯化硅）
当前状态: 多头排列 + 均线发散 + 距MA60仅20%

与长飞光纤的相似性:
  ✅ 同一产业链: 长飞做光纤预制棒，宏柏做上游高纯四氯化硅
  ✅ 光纤涨价周期: 下游需求爆发 → 上游材料供不应求
  ✅ 技术面位置: 距MA60仅20%，类似长飞2025年7月的位置
  ✅ 均线刚发散: MA5>MA10>MA20，趋势启动初期

长飞模式映射:
  如果光纤涨价周期持续2-3年，宏柏作为上游材料商
  可能复制长飞的涨幅路径（当然幅度不一定相同）

风险:
  ⚠️ 市值较小，流动性不如长飞
  ⚠️ 含硫硅烷价格波动大
  ⚠️ 需验证Q2业绩是否兑现涨价
```
