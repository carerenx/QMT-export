# "前期发现"选股策略 V1.0

> **策略目标**: 在全市场主板上，通过技术面+资金面多维度评分，发现处于**类似长飞光纤(601869)2025年中期早期启动阶段**的潜力标的
> **适用市场**: A股主板（上海60xxxx + 深圳00xxxx），剔除创业板/科创板/北交所
> **更新频率**: 每日收盘后或每周一次
> **依赖数据源**: 同花顺热点API / 东财板块排名API / Tencent K线API / 腾讯财经行情API

---

## 一、策略核心理念

### 1.1 不追高，找"下一个"

长飞光纤在2025年6-7月的特征是可以被识别和量化的：

```
2025年6月长飞光纤的状态:
  价格: ~40元
  趋势: 刚开始站上MA60，均线初现多头排列
  位置: 距MA60约15%（极早期）
  量能: 成交量温和放大（1.5x均量）
  动量: 稳步上行（非急涨）
  估值: PE尚可接受
  板块: 通信/光纤，有基本面支撑
  
2026年6月长飞光纤的状态:
  价格: ~543元
  趋势: 极度延伸
  位置: 距MA60超过100%（严重透支）
  PE: 388x（泡沫区间）
  
  → 此时已不是"前期发现"，而是"追高博傻"
```

### 1.2 策略哲学

```
不在暴涨后追高 → 在同样的早期信号出现时找到下一个
不靠单一指标 → 多维度交叉验证
不预测涨幅 → 只判断"是否处于早期有利位置"
宁可错过 → 不可做错（高评分才出手）
```

## 二、数据采集流程

### 2.1 步骤总览

```
Step 1: 收集近期强势股（同花顺热点API，最近8天）
Step 2: 收集成长行业龙头（东财板块排名，17个行业）
Step 3: 过滤至主板（剔除300/301/688/689/8xx/4xx）
Step 4: 拉取实时行情（腾讯财经API，获取PE/PB/量比等）
Step 5: 拉取60日K线（腾讯K线API，逐只计算技术指标）
Step 6: 多维度评分（趋势/位置/量能/动量/估值）
Step 7: 排名 → 人工过滤ST → 输出推荐列表
```

```python
from datetime import date, timedelta
import requests

UA = 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) Chrome/117.0.0.0 Safari/537.36'
hot_stocks = {}  # code -> {name, tags[], count, total_gain}

for i in range(8):
    d = date.today() - timedelta(days=i+1)
    ds = d.strftime('%Y-%m-%d')
    url = f'http://zx.10jqka.com.cn/event/api/getharden/date/{ds}/orderby/date/orderway/desc/charset/GBK/'
    r = requests.get(url, headers={'User-Agent': UA}, timeout=10)
    data = r.json()
    if data.get('errocode', 0) != 0: continue
    rows = data.get('data') or []
    
    for row in rows:
        code = row.get('code', '')
        reason = row.get('reason', '') or ''
        tags = [t.strip() for t in reason.split('+') if t.strip()]
        
        if code not in hot_stocks:
            hot_stocks[code] = {'name': row.get('name', ''), 'tags': [], 'count': 0}
        hot_stocks[code]['count'] += 1
        for t in tags:
            if t not in hot_stocks[code]['tags']:
                hot_stocks[code]['tags'].append(t)

# 过滤: 至少出现2天（持续强势，排除一日游）
persistent = {c: s for c, s in hot_stocks.items() if s['count'] >= 2}
```

### 2.5 Step 4 — 拉取实时行情

```python
import urllib.request

def fetch_quotes(codes):
    """批量拉取腾讯财经实时行情，每批80只"""
    quotes = {}
    for i in range(0, len(codes), 80):
        batch = codes[i:i+80]
        prefixed = [f'sh{c}' if c.startswith(('6','9')) else f'sz{c}' for c in batch]
        url = 'https://qt.gtimg.cn/q=' + ','.join(prefixed)
        req = urllib.request.Request(url)
        req.add_header('User-Agent', UA)
        resp = urllib.request.urlopen(req, timeout=15)
        data = resp.read().decode('gbk')
        
        for line in data.strip().split(';'):
            if '=' not in line or '"' not in line: continue
            vals = line.split('"')[1].split('~')
            if len(vals) < 53: continue
            code = line.split('=')[0].split('_')[-1][2:]
            quotes[code] = {
                'name': vals[1], 'price': float(vals[3]) if vals[3] else 0,
                'pe_ttm': float(vals[39]) if vals[39] else 0,
                'pb': float(vals[46]) if vals[46] else 0,
                'mcap_yi': float(vals[44]) if vals[44] else 0,
                'change_pct': float(vals[32]) if vals[32] else 0,
                'turnover_pct': float(vals[38]) if vals[38] else 0,
                'vol_ratio': float(vals[49]) if vals[49] else 0,
                'pe_static': float(vals[52]) if vals[52] else 0,
            }
        time.sleep(0.5)
    return quotes
```

### 2.6 Step 5 — 拉取K线并计算技术指标

```python
def get_kline_and_score(code):
    """拉取60日K线，计算指标并评分"""
    tc = f'sh{code}' if code.startswith(('6','9')) else f'sz{code}'
    
    # 1. 拉取K线
    url = 'https://web.ifzq.gtimg.cn/appstock/app/fqkline/get'
    r = requests.get(url, params={'param': f'{tc},day,,,60,qfq'},
                    headers={'User-Agent': UA, 'Referer': 'https://gu.qq.com/'}, timeout=8)
    d = r.json()
    raw = d.get('data', {}).get(tc, {}).get('qfqday', []) or \
          d.get('data', {}).get(tc, {}).get('day', [])
    if not raw or len(raw) < 30:
        return None
    
    # 2. 提取数据
    closes = [float(k[2]) for k in raw]
    highs = [float(k[3]) for k in raw]
    lows = [float(k[4]) for k in raw]
    volumes = [float(k[5]) for k in raw]
    current = closes[-1]
    
    # 3. 计算均线
    ma5 = sum(closes[-5:]) / 5
    ma10 = sum(closes[-10:]) / 10
    ma20 = sum(closes[-20:]) / 20
    ma60 = sum(closes[-60:]) / 60 if len(closes) >= 60 else ma20
    
    # 4. 计算关键指标
    pct_from_ma20 = (current - ma20) / ma20 * 100
    pct_from_ma60 = (current - ma60) / ma60 * 100
    vol_avg20 = sum(volumes[-20:]) / 20
    vol_recent = sum(volumes[-5:]) / 5
    vol_ratio = vol_recent / vol_avg20 if vol_avg20 > 0 else 1
    chg_20d = (current - closes[-21]) / closes[-21] * 100 if len(closes) > 21 else 0
    high_10 = max(highs[-10:])
    dd_10d = (current - high_10) / high_10 * 100
    high_20 = max(highs[-20:])
    
    return {
        'ma5': ma5, 'ma10': ma10, 'ma20': ma20, 'ma60': ma60,
        'pct_ma20': pct_from_ma20, 'pct_ma60': pct_from_ma60,
        'vol_ratio': vol_ratio, 'chg_20d': chg_20d, 'dd_10d': dd_10d,
        'is_new_high': current >= high_20 * 0.97,
        'ma_bullish': ma5 > ma10 > ma20,
        'trend_aligned': current > ma20 > ma60,
        'early_stage': 0 < pct_from_ma60 < 45,
    }
```

## 三、评分模型

### 3.1 评分维度与权重

| 维度 | 满分 | 核心逻辑 |
|------|------|----------|
| **趋势结构** | 35分 | 多头排列是超级牛股的基础 |
| **早期位置** | 25分 | 距MA60越近越好（未透支） |
| **量能健康** | 15分 | 温和放量说明资金在有序介入 |
| **动量节奏** | 15分 | 稳健上行优于急涨 |
| **估值安全** | 10分 | PE合理提供基本面保护 |
| **总计** | **100分** | |

### 3.2 详细评分规则

```python
def score_stock(tech, quote):
    """
    对一只股票进行综合评分
    
    参数:
      tech: get_kline_and_score() 的返回结果
      quote: 腾讯财经实时行情数据
    
    返回:
      score: 0-100分
      reasons: 触发的加分理由列表
    """
    score = 0
    reasons = []
    
    # ===== 第一部分: 趋势结构 (满分35分) =====
    
    # 1.1 多头排列 (20分)
    if tech['trend_aligned']:
        score += 20
        reasons.append('多头排列(价>MA20>MA60)')
    elif tech['ma5'] > tech['ma20']:
        score += 12
        reasons.append('价格站上MA20')
    
    # 1.2 均线发散 (12分)
    if tech['ma_bullish']:
        score += 12
        reasons.append('均线发散向上(MA5>MA10>MA20)')
    elif tech['ma5'] > tech['ma20']:
        score += 6
    
    # 1.3 近期突破确认 (3分) - 加分项
    if tech['is_new_high']:
        score += 3
        reasons.append('近期创新高')
    
    # ===== 第二部分: 早期位置 (满分25分) =====
    
    pct60 = tech['pct_ma60']
    
    if 0 < pct60 <= 15:
        score += 25
        reasons.append(f'极早期(距MA60仅{pct60:.0f}%)')
    elif 15 < pct60 <= 25:
        score += 22
        reasons.append(f'早期(距MA60={pct60:.0f}%)')
    elif 25 < pct60 <= 35:
        score += 17
        reasons.append(f'早期偏中(距MA60={pct60:.0f}%)')
    elif 35 < pct60 <= 45:
        score += 10
        reasons.append(f'中期(距MA60={pct60:.0f}%)')
    elif 45 < pct60 <= 65:
        score += 5
        reasons.append(f'中期偏后(距MA60={pct60:.0f}%)')
    elif pct60 > 100:
        score -= 8
        reasons.append(f'高位(距MA60={pct60:.0f}%,已透支)')
    
    # ===== 第三部分: 量能健康 (满分15分) =====
    
    vr = tech['vol_ratio']
    
    if 1.2 < vr <= 1.8:
        score += 15
        reasons.append(f'温和放量({vr:.1f}x均量)')
    elif 1.8 < vr <= 2.5:
        score += 11
        reasons.append(f'明显放量({vr:.1f}x均量)')
    elif 1.0 < vr <= 1.2:
        score += 8
        reasons.append('量能正常')
    elif vr > 3.0:
        score += 3
        reasons.append(f'剧烈放量({vr:.1f}x,需警惕过热)')
    
    # ===== 第四部分: 动量节奏 (满分15分) =====
    
    chg20 = tech['chg_20d']
    
    if 5 < chg20 <= 15:
        score += 15
        reasons.append(f'稳健上行(20日+{chg20:.0f}%)')
    elif 15 < chg20 <= 25:
        score += 12
        reasons.append(f'温和上涨(20日+{chg20:.0f}%)')
    elif 25 < chg20 <= 45:
        score += 7
        reasons.append(f'加速中(20日+{chg20:.0f}%)')
    elif chg20 > 60:
        score -= 3
        reasons.append(f'短期过热(20日+{chg20:.0f}%)')
    
    # 回撤控制 (在"早期位置"已含，这里做微调)
    if tech['dd_10d'] > -5:
        score += 3
        reasons.append('回撤极小(筹码锁定好)')
    elif tech['dd_10d'] > -8:
        score += 1
    
    # ===== 第五部分: 估值安全 (满分10分) =====
    
    pe = quote.get('pe_ttm', 0)
    
    if 0 < pe <= 25:
        score += 10
        reasons.append(f'PE={pe:.0f}x(低估)')
    elif 25 < pe <= 40:
        score += 8
        reasons.append(f'PE={pe:.0f}x(偏低)')
    elif 40 < pe <= 60:
        score += 5
        reasons.append(f'PE={pe:.0f}x(合理)')
    elif 60 < pe <= 90:
        score += 2
        reasons.append(f'PE={pe:.0f}x(偏高)')
    elif pe > 200:
        score -= 5
        reasons.append(f'PE={pe:.0f}x(极度高估)')
    elif pe <= 0:
        score -= 3  # 亏损股扣分
    
    return score, reasons
```

### 3.3 评分等级

| 分数 | 等级 | 操作建议 |
|------|------|----------|
| **≥80** | ⭐⭐⭐⭐⭐ 极佳 | 重点关注，深度研究基本面 |
| **65-79** | ⭐⭐⭐⭐ 良好 | 加入观察列表，等待回调买点 |
| **50-64** | ⭐⭐⭐ 一般 | 保持关注，等待技术面改善 |
| **35-49** | ⭐⭐ 偏弱 | 暂不关注 |
| **<35** | ⭐ 差 | 过滤掉 |

## 七、已知局限与改进方向

### 7.1 当前局限

| 局限 | 影响 | 改进方向 |
|------|------|----------|
| 仅看技术面 | 可能选中基本面有问题的股票 | 加入财报数据（营收增速/利润增速） |

## 八、策略声明

> **核心原则**: 本策略旨在发现"处于早期启动阶段"的标的，不保证任何股票能复制长飞光纤的涨幅。长飞+1300%是多重周期共振的极端案例，不具备普遍性。
>
> **风险提示**: 技术面筛选只是第一步，实盘交易前必须进行基本面验证。任何策略都有失效的时候，严格遵守止损纪律是生存的第一法则。
>
> **数据来源**: 同花顺热点API / 东财push2 API / 腾讯财经API / 腾讯K线API，均免费无需API Key。
