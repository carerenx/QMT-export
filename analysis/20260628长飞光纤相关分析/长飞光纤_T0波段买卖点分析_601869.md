| 资金大额流出 | 19次 | **+7.6%** | 79% |
| 远离MA20(>15%) | 85次 | **+6.6%** | 62% |

> **关键数据**: 在长飞光纤上，所有传统的"卖出信号"出现后，5天内股价**平均继续上涨+6~9%**。这意味着任何基于传统超买指标的卖出都会卖飞。

### 1.2 做T的根本原则

```
在超级强势股中做T的唯一正确姿势:

  ✅ 回调到支撑位 → 加仓买入（低吸）
  ✅ 反弹到阻力位 → 卖出加仓部分（高抛保留底仓）
  ✅ 永远保留底仓（50-70%）不动 → 让主升浪利润奔跑
  
  ❌ 清仓式卖出 → 一定会卖飞
  ❌ 追高买入 → 短期被套但趋势向上会解套
  ❌ 在"超买"信号下全部离场 → 后面的涨幅与你无关
```

## 三、精确卖点：何时减仓

### 3.1 不要全卖！做T的核心是"卖加仓部分"

```
正确操作:
  底部买入1000股 → 反弹到阻力位卖出600股 → 保留400股底仓
  这样你既锁定了利润，又不会完全踏空后续涨幅
```

### 3.2 卖出信号A：RSI极端超买（减仓50-70%的加仓部分）

```
条件:
  ① RSI(14) > 78              → 极端超买（比常规70阈值更高）
  ② 浮动盈利 > 3%              → 已有利润可锁定
  ③ 成交量比 > 1.3             → 放量拉升（可能是短期高潮）

卖出比例: 加仓部分的50-70%
保留: 底仓不动
```

### 3.3 卖出信号B：布林上轨放量滞涨（减仓80%加仓部分）

```
条件:
  ① 布林带位置 > 90%           → 触及或接近上轨
  ② 成交量比 > 1.5             → 明显放量
  ③ 收盘 < 开盘（收阴线）      → 高位滞涨信号
  ④ 浮动盈利 > 0               → 至少不亏

卖出比例: 加仓部分的80%
保留: 底仓不动
```

### 3.4 卖出信号C：移动止盈（最后防线）

```
条件:
  从买入后最高点回落 > 7% 且浮动盈利 > 5%
  
  例如: 420买入 → 涨到490(最高) → 跌到455 → 触发止盈
  此时卖出加仓部分，锁定+8.3%利润
  
保留: 底仓不动（除非MA60破位才考虑减底仓）
```

### 3.5 绝对不要卖的信号（重要！）

以下信号在长飞光纤上都是**假信号**，触发后股价继续上涨：

```
❌ RSI>70就卖 → 卖后平均还涨6.1%
❌ MACD死叉就卖 → 卖后平均还涨7.8%
❌ 资金流出就卖 → 卖后平均还涨7.6%
❌ 远离MA20就卖 → 卖后平均还涨6.6%
❌ 布林上轨就卖 → 卖后平均还涨9.1%

这些"经典卖出信号"在超级牛股中全部失效。
只有RSI>78+收阴+放量三个条件同时满足时，才考虑减仓（而非清仓）。
```
