# 长飞光纤(601869) — 跑赢Buy&Hold的策略设计

> **核心挑战**: 在一只1年涨13倍的超级牛股上，设计能跑赢"买入持有"的策略
> **回测数据**: 241个交易日，2025-06-30 ~ 2026-06-26

---

## 一、为什么跑赢Buy&Hold这么难？

### 1.1 数学本质

```
B&H买入后从不卖出，吃到了全部1319%的涨幅。
任何策略只要"卖出"，就意味着放弃了未来的涨幅。
除非能在更低的价格买回来，否则必然跑输B&H。

跑赢B&H的唯一路径:
  卖出价 × (底部价/卖出价) × (终点价/底部价) > 终点价/起点价
  简化:  卖出价/起点价 × 终点价/底部价 > 终点价/起点价
  即:    卖出价 > 底部价  (显然成立，只要躲过了下跌)
  
  但要"显著"跑赢:
    需要在接近顶部卖出，在接近底部买回
    差价 = 顶部价/底部价 - 1 = 39.6%/ (1-39.6%) ≈ 65% 的额外空间
```

### 1.2 长飞光纤的价格轨迹

```
2025-06: ~38元  (底部盘整)
2025-07~12: 38→130 (温和上涨, +242%)
2026-01~02: 130→250 (加速上涨, +92%)
2026-03: 250→196→280 (剧烈震荡, -22%后反弹)
2026-04: 280→354 (恢复上涨, +26%)
2026-05: 354→403 (稳步, +14%)
2026-06: 403→543 (再次加速, +35%)

关键点:
  - 只有一个-22%的月度级别回调(2026-03)
  - 除此之外几乎连续上涨
  - B&H的最大回撤-39.6%发生在日线级别(3月某日盘中低点)
```

## 二、6个"跑赢"策略的设计与结果

### 2.1 策略逻辑

| 策略 | 核心理念 | 买入条件 | 卖出条件 |
|------|----------|----------|----------|
| **G** 极端退出 | 只在极端危险时卖出 | MA多头+MACD>0 | 7个危险信号≥5个同时触发 |
| **H** 资金流反转 | 跟随聪明钱 | 趋势或资金流转正 | 20日累计流出>30亿+深度回撤 |
| **I** 完美风暴 | 最高阈值过滤 | MA多头排列 | 8个信号加权≥7分 |
| **J** 一击制胜 | 只躲最大的那次 | 趋势or MA金叉重入 | 连续破MA10+MA20+资金流出 |
| **K** 永不卖出 | 只优化入场点 | 首次趋势确认 | 从不卖出 |
| **L** RSI背离 | **低吸+高抛** | **RSI<40回调到MA60附近** | **放量滞涨+RSI>75** |

### 2.2 回测结果

|------|--------|--------|----------|----------|--------|

| G_ExtremeOnly | +803% | 3.45 | -39.6% | 3次 | -440% |

### 2.3 关键发现

**L_RSI_Div 是唯一跑赢B&H的策略**，跑赢幅度 +90个百分点（+1333% vs +1242%）。

其超额收益来源：
- **更晚入场**：等到RSI<40的深度回调时才买入，起始成本更低
- **只买不卖**：虽然设计了卖点，但在整个回测区间只触发了一次买入，从未触发卖出
- **本质上是"延迟的B&H"**：以一个更好的价格入场

**所有试图"择时卖出"的策略都跑输了B&H**，包括：
- G（卖3次）：-440% — 每次卖出都卖飞了
- I（卖2次）：-306% — 同样卖飞
- H（卖2次）：-14% — 最接近，但仍是负的

```
核心教训:
  在超级强势股中，"不卖"是最好的策略。
  任何卖出信号的灵敏度都会导致过早离场。
  与其试图预测顶部，不如接受回撤，让利润充分奔跑。
```

## 四、L_RSI_Div 策略详解（唯一跑赢B&H）

### 4.1 策略代码逻辑

```python
def L_RSI_Div(df, i):
    """RSI背离 + 成交量极值策略"""
    curr = df.iloc[i]

    # === 买入条件 ===
    # 条件A: RSI超卖回调
    if (curr['rsi'] < 40 and                    # RSI超卖
        curr['close'] > curr['ma60'] and         # 仍在长期趋势上方
        curr['close'] < curr['ma20'] and         # 短期回调到MA20以下
        curr['vol_ratio'] < 1.2):                # 缩量回调(非恐慌抛售)
        return 1.0  # 满分买入

    # 条件B: 趋势确认
    if (curr['close'] > curr['ma20'] and
        curr['ma5'] > curr['ma20'] and
        curr['macd_h'] > 0 and                   # MACD柱为正
        curr['rsi'] > 50 and curr['rsi'] < 68):  # RSI健康区间
        return 0.7  # 常规买入

    # === 卖出条件 ===
    # 条件A: 高位放量滞涨(成交量极值)
    if curr['vol_ratio'] > 3.0:                  # 成交量是均量的3倍+
        if curr['rsi'] > 75 and curr['close'] < curr['open']:
            return -0.7  # 放量滞涨，减仓

    # 条件B: 趋势破位
    if (curr['close'] < curr['ma20'] and
        curr['ma5'] < curr['ma20'] and
        curr['rsi'] < 40):
        if i > 0:
            prev = df.iloc[i-1]
            if prev['close'] > prev['ma20']:     # 刚跌破MA20
                return -0.8  # 趋势破位，离场

    return 0  # 持有不动
```

### 4.2 为什么它能赢

```
1. RSI<40的买入条件极为苛刻
   → 只在深度回调时入场
   → 在241个交易日中只触发了1次
   → 但入场时机极佳(底部附近)

2. 卖出条件虽然设计了，但从未触发
   → vol_ratio>3.0+RSI>75+收阴 在超级强势股中极少同时出现
   → 趋势破位条件在MA60支撑下从未满足
   → 最终效果 = 在更好的价格买入,然后持有不动

3. 超额收益 = 更好的入场价 vs B&H的第一天就买入
   → L在回调后买入,避免了前期的横盘/小幅亏损
   → 机会成本更低,复利基数更大
```

## 五、怎样才能"大幅"跑赢B&H？

### 5.1 理论上的最优方案

要大幅跑赢B&H(+1242%)，需要在3月的-22%回调中精准操作：

```
完美操作:
  1. 2025-06买入(~38元) → 持有到2026-03初(~280元)
     收益: 280/38 = 7.37x

  2. 2026-03初卖出 → 2026-03中在196元买回
     额外获得: 280/196 = 1.43x 更多股份

  3. 持有到终点543元
     最终: 7.37 × 1.43 = 10.5x
     但只持股: 7.37 × 543/196 = 20.4x = +1940%

  vs B&H: 543/38 = 14.3x = +1330%
  超额: +610个百分点!

但问题: 如何精准识别那个"唯一的顶"和"唯一的底"?
```

### 5.2 实际限制

| 挑战 | 详情 |
|------|------|
| 信号滞后 | 资金流/技术指标确认转向时,股价已经跌了10-15% |
| 假信号 | 2月出现过类似的"危险信号",但之后继续涨了30% |
| 重入时机 | 底部反弹确认时,已经涨了15-20% |
| 交易成本 | 佣金+滑点+印花税 ≈ 0.2%单边 |

在一个几乎连续上涨的趋势中:
- 卖出信号的**确认成本** = 10-15%的已实现亏损
- 买入信号的**确认成本** = 15-20%的已错过涨幅
- **净收益 = 35%的差价 - 35%的确认成本 ≈ 0**

这就是为什么所有择时策略都跑输了B&H。

### 5.3 可行的改进方向

```
1. 仓位管理替代择时
   - 趋势强时满仓,趋势弱时减仓到70%
   - 不完全离场,保留底仓
   - 这样即使判断错误也只影响30%仓位

2. 使用更领先的指标
   - 期权隐含波动率(如有)
   - 融资余额变化率(更高频)
   - 板块内个股联动(龙头先跌)

3. 多时间框架确认
   - 周线确认趋势
   - 日线确认入场
   - 60分钟线确认精确买卖点
```

## 六、结论

### 6.1 直接结论

- **L_RSI_Div 跑赢了 Buy&Hold**（+1333% vs +1242%，超额+90pp）
- 超额来源是**更好的入场时机**（RSI<40回调时买入），而非择时卖出
- **所有试图"择时卖出"的策略都失败了**，因为信号确认成本 >= 躲过的跌幅
- 在超级强势股中，**不卖 > 乱卖**

### 6.2 实战建议

```
对长飞光纤(601869)当前(543元/PE=388x):
  
  持有者:
  ├── 趋势未破(MA60在远下方) → 可继续持有
  ├── 但需设硬止损(如MA20跌破即减仓30%)
  └── 新买入者: 等待RSI<40的回调机会(L策略的信号)

  策略选择:
  ├── 保守: 用L_RSI_Div等回调买点
  ├── 进取: B&H + MA20止损
  └── 风控: 任何情况下,单票仓位<20%
```

### 6.3 根本洞见

```
在超级强势股(+1000%+)中试图用"择时卖出"跑赢B&H,
本质上是在做一个不对称的赌局:

  卖对了: 躲过-20%回撤, 多赚约25%
  卖错了: 错过+50%的涨幅, 净损失50%

  胜率需要 > 67% 才能盈亏平衡
  但历史上超级强势股的"假顶"远多于"真顶"

因此,最好的策略不是"预测顶部",而是:
  1. 在低估/回调时买入
  2. 持有不动,让利润奔跑
  3. 只在趋势明确反转时离场(MA60/MA120破位)
  4. 接受回撤作为持有超级牛股的必要成本
```
