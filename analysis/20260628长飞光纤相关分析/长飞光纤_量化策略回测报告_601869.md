## 三、策略设计

针对长飞光纤"强势上涨+高波动"的特征，设计了**7个量化策略**：

| 策略 | 名称 | 买入逻辑 | 卖出逻辑 |
|------|------|----------|----------|
| **S1** | MA双均线 | MA5金叉MA20 | MA5死叉MA20 |
| **S2** | 布林带反转 | 价格触下轨+RSI<40 | 价格触上轨+RSI>60 |
| **S3** | 量价突破 | 放量1.5x+突破MA20+RSI40~75 | 放量2.2x+跌破MA10 |
| **S4** | 趋势跟踪+移动止盈 | MA多头排列+放量+MACD>0 | ATR跟踪止损/MA10死叉 |
| **S5** | MACD动量 | MACD柱金叉+价格>MA20 | MACD柱死叉 |
| **S6** | 综合融合 | S1*0.25+S2*0.15+S3*0.35+S5*0.25 | 加权得分≤-0.5 |
| **S7** | 资金流跟踪 | 5日主力净流入+价格>MA20 | 3日主力大额流出+跌破MA10 |

### 关键发现

- **没有任何策略跑赢 Buy & Hold**——这在一只1年涨13倍的股票上是必然的。任何卖出都会错失后续涨幅。
- **S3量价突破(1102%)最接近Buy&Hold**——因为它只在最强趋势启动时入场，之后从未触发卖出信号。
- **S5 MACD动量(510%)是多交易策略中最优**——9次交易、78%胜率、Sharpe 3.43、最大回撤仅16.9%。
- **S4趋势跟踪(352%)风险调整后最优**——Calmar比率23.2，回撤控制在16.1%。
- **S2布林带(8.4%)完全失效**——在超级强势股中，价格几乎从不触及下轨，买入机会极少。

## 九、核心结论

### 9.1 关于长飞光纤

1. **超级强势股的特殊性**: 长飞近1年涨1319%，几乎没有任何有效回调能触发卖出。所有策略都跑输Buy&Hold，因为**任何卖出都是错的**。

2. **但这不可持续**: PE=388x + 主力-121亿流出，当前的涨势本质上是"博傻"——每个人都相信还有更傻的人接盘。

3. **策略真正的价值**: 回测的目的不是为了在这只股票上赚钱，而是为了**在类似股票上**有一套可重复的系统。当趋势终结时，S4的趋势跟踪止损能保护利润。

### 9.2 推荐的实战方案

```
方案A（进攻型）: S5 MACD动量
  - 9次交易，78%胜率，Sharpe 3.43
  - 每次持仓约15天，适合活跃交易者
  - 风险: 在最长的主升浪中可能过早卖出（如第5笔仅+0.1%）

方案B（防守型）: S4 趋势跟踪+移动止盈
  - 7次交易，71%胜率，最大回撤仅16.1%
  - ATR跟踪止损保护利润，适合大资金
  - 风险: 假突破时可能小额亏损（如第5笔-0.5%）

方案C（组合型）: S5(60%) + S4(40%)
  - 兼顾进攻与防守
  - 在强势趋势中让利润奔跑
  - 在趋势转弱时保护本金
```

### 9.3 策略参数优化建议

```
S5 MACD动量优化方向:
  1. 增加成交量确认: 买入时要求 volume_ratio > 1.2
  2. 增加趋势过滤: 买入时要求 close > MA60
  3. 增加止盈: 单笔盈利>30%时，启动移动止盈（ATR×2）

S4 趋势跟踪优化方向:
  1. ATR倍数调优: 当前2.5x，可测试2.0x~3.0x
  2. 增加再入场逻辑: 止盈后若价格重新突破MA20可再次入场
  3. 增加市场环境过滤: 在大盘下跌时降低仓位
```
