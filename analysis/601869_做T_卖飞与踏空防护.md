# 长飞光纤(601869) 日内做T —— 卖飞与踏空防护机制

> 做T策略最核心的两个风险：卖飞（卖太早，少赚）和踏空（没买到，错过）。
> 本文把每种风险的触发场景、严重程度、防护机制一一拆开。

---

## 一、风险全景图

```
                    做T当天股价走势

  涨 ▲                       卖飞区
     │                    ╱   (反T卖早了，买不回来)
     │              ╱──╱
     │         ╱──╱  ← 正T第二次卖点(如果卖了就卖飞)
     │    ╱──╱
     │ ╱─╱  ← 正T第一次卖点
     │╱
 开盘├────────────────────────────→
     │╲
     │  ╲  ← 正T买入点
     │    ╲──╲
     │         ╲──╲  ← 踏空区 (等更低，没等到)
     │              ╲──╲
     │                    ╲  踏空区
  跌 ▼                        (反T没在高位卖掉)
```

### 卖飞 vs 踏空：哪个更痛？

| | 卖飞（Sell Too Early） | 踏空（Miss Entry） |
|---|---|---|
| **发生场景** | 正T：卖了之后继续涨<br>反T：卖了底仓后买不回来 | 正T：等回调，结果一路涨<br>反T：等反弹，结果一路跌 |
| **直接损失** | 少赚（机会成本） | 完全没赚（0收益） |
| **心理伤害** | ⭐⭐⭐⭐⭐ 极大 | ⭐⭐⭐ 中等 |
| **对长飞的影响** | 牛市中最常见（强趋势+高波动） | 加速上涨日最常见 |
| **发生频率** | 高（约30-40%的做T日） | 中（约20-30%的交易日） |

> **对长飞光纤而言，卖飞是更大更频繁的风险。** 原因是长飞处于强牛市中，日内走势以"低开高走"和"探底回升"为主——正T买入后如果太早卖出，容易错过后半程的主升。

## 二、卖飞：五种场景 + 五种防护

### 场景1：正T买在最低点附近，卖在第一个阻力位，但股价继续冲高

```
例：开盘500 → 跌到485（买入）→ 反弹到498（卖出，+2.7%）→ 继续涨到520（卖飞，少赚4.5%）
```

**防护1：分批止盈 (Staggered Take-Profit)**

```
做T买入的股数分3批卖出：

批次    占比    卖出条件              适用场景
─────  ──────  ────────────────────  ──────────
第1批   40%    触及第一目标(+1.5ATR)  锁定基础利润
第2批   35%    触及第二目标(+2.5ATR)  捕捉主升浪
第3批   25%    移动止盈或收盘强制平仓  让利润奔跑

效果：至少40%的T仓位吃到了基础利润，25-60%有机会吃到超预期涨幅
```

**防护2：移动止盈 (Trailing Stop)**

```python
# 伪代码
if 正T已买入:
    初始止损 = 买入价 * (1 - STOP_LOSS_PCT)    # 硬止损-3%
    移动止盈触发价 = 买入价 * (1 + 1.5*ATR%)   # 盈利超过1.5个ATR后启动
    
    if 当前价 > 移动止盈触发价:
        移动止盈价 = 当前价 * (1 - 0.5*ATR%)   # 回撤0.5个ATR就止盈
        # 如果股价继续涨，移动止盈价跟着往上移
        # 如果股价回撤超过0.5ATR，触发止盈
```

### 场景2：反T在牛市中卖飞——最致命的卖飞

```
例：开盘500 → 冲高到515（卖出底仓）→ 期望回调到505买回
    → 结果股价一路上涨到530（卖飞，每股亏15元，且丢了底仓）
```

**这是做T策略最大的风险。丢了底仓 = 丢了整个牛市的仓位。**

**防护3：牛市反T熔断 (Bull Market Short-T Circuit Breaker)**

```python
# 反T的"三重门"——必须同时满足才允许反T

条件1: 趋势判断不是牛市 (MA5 > MA20 AND close > MA20 → 不允许反T)
条件2: 当日开盘涨幅 < 2% (大幅高开说明强势，不反T)
条件3: 连涨天数 < 3 (连涨3天说明加速中，不反T)
条件4: MACD柱 > 0 (MACD为正说明动能向上，谨慎反T)

只有四个条件全部通过，才允许反T。
```

**防护4：反T买回时限 (Time-Limited Buyback)**

```python
# 反T卖出后，必须在30分钟内买回
# 如果股价继续上涨超过卖出价+1%，立即止损买回

if 反T已卖出:
    最大容忍上涨 = 卖出价 * 1.01   # 如果涨超1%，认赔买回
    if 当前价 > 最大容忍上涨:
        立即买回  # 止损反T，保护底仓
```

### 场景3：正T买入后，股价横盘不涨，收盘被迫平仓

```
这不是卖飞，而是"做了也白做"。但频繁发生会消耗手续费。
```

**防护5：时间衰减平仓 (Time-Decay Exit)**

```python
if 正T已买入 and 持仓时间 > 2小时 and 浮盈 < 0.3%:
    主动平仓  # 不赚钱就早点走，省手续费、释放资金
```

## 三、踏空：三种场景 + 三种防护

### 场景1：正T踏空——等回调，但股价一路涨

```
例：开盘500 → 设买入价495 → 结果最低只到498 → 没买到 → 收盘520
    踏空损失：20元/股的机会
```

**防护A：阶梯挂单 (Laddered Buy Orders)**

```python
# 不要把鸡蛋放在一个篮子里——在3个价格水平挂买单

层级     挂单价             股数占比   逻辑
──────  ──────────────────  ────────  ──────────────
第1层   开盘价 - 0.3*ATR    30%      浅回调（概率高，利润小）
第2层   开盘价 - 0.6*ATR    40%      中等回调（核心仓位）
第3层   开盘价 - 1.0*ATR    30%      深回调（安全边际大，但概率低）

效果：只要股价有回调，至少有一层能成交
```

### 场景2：大幅高开踏空——开盘直接跳空高开5%，不敢追

```
例：昨收480 → 今开510（+6.2%）→ 觉得太高不敢买 → 收盘550
    踏空损失：40元/股
```

**防护B：跳空追入策略 (Gap Chase Strategy)**

```python
# 不是所有跳空都是陷阱——区分"强势跳空"和"衰竭跳空"

跳空幅度    成交量条件              操作
──────────  ──────────────────────  ────────────────
+2% ~ +4%   量比 > 1.5 AND 主力净流入  小仓位追入(10%底仓)
+4% ~ +7%   量比 > 2.0 AND MACD向上     极轻仓追入(5%底仓)
> +7%       —                        不做T，观望
< +2%       正常                      正常阶梯挂单

# 追入后止损设在开盘价下方0.5ATR
```

### 场景3：尾盘踏空——全天没触发买入条件，尾盘突然拉升

```
这是最常见的踏空形态，也是"最难防范"的——因为尾盘拉升往往没有预兆。
```

**防护C：尾盘补偿机制 (Closing Hour Catch-up)**

```python
if 全天未触发T买入 AND 现在时间 > 14:30:
    if 股价 > 均价 AND 量比 > 1.2 AND 主力净流入 > 0:
        # 尾盘买入少量，博次日高开
        尾盘T买入(底仓的10%)
        # 次日开盘立即卖出（隔夜T）
        标记为隔夜T仓位
```

## 七、最终结论

> **做T策略中，卖飞比踏空更致命。**
>
> 踏空 = 今天没赚到，明天还有机会。卖飞（尤其是反T卖飞）= 丢了底仓，可能永远买不回来。
>
> **核心防护原则**：
> 1. **牛市永远不做反T** —— 这是铁律，违反就可能在单日丢底仓
> 2. **买入用阶梯，不要用单点** —— 大幅降低踏空概率
> 3. **卖出用分批，不要一口吃** —— 让一部分仓位去博超预期涨幅
> 4. **反T卖出后必须设硬止损买回** —— 涨超1%立即认赔买回，保护底仓
> 5. **尾盘不贪，收盘必须平** —— 日内T不过夜是纪律

### 最佳实践组合

| 市场状态 | 做T方向 | 买入方式 | 卖出方式 | 仓位上限 |
|----------|--------|---------|---------|---------|
| **强牛市** (当前) | 只做正T | 阶梯3层 | 分批+移动止盈 | 底仓30% |
| 震荡 | 正T+反T | 阶梯3层 | 分批止盈 | 底仓40% |
| 弱熊市 | 只做反T | 阶梯3层(卖) | 分批买回+硬止损 | 底仓20% |
| 急跌/暴涨 | 不做T | — | — | 0% |
