## 一、策略设计理念

### 1.1 为什么长飞光纤适合做T？

从我们的 [Serenity 深度研究](serenity_challenge_601869.md) 中提炼的关键特征：

| 特征 | 数值 | 做T适配性 |
|------|------|----------|
| **高价股** | ~400-520元/股 | ★★★★★ 单笔波动绝对值大，T收益弹性高 |
| **高波动** | 日均振幅 5.54%，ATR均值 7.59元 | ★★★★★ 日内价差空间充足 |
| **强趋势** | AI光纤超级周期，上涨趋势明确 | ★★★★☆ 顺势做T胜率高 |
| **流动性好** | 日均成交数十亿 | ★★★★☆ 买卖价差小，滑点可控 |
| **机构参与深** | 沪股通+QFII+公募重仓 | ★★★☆☆ 盘中波动有规律可循 |

**做T核心逻辑**：长飞光纤日均振幅超5%，意味着每100万元底仓每天有~5万元的价格波动空间。通过量化择时捕捉其中1-2%的确定性波段，可实现稳定的日频增强收益。

### 1.2 A股做T的T+1约束

```
T+1制度下，当日买入的股票当日不可卖出。
但如果你已持有底仓（前一日已买入的股票），可以做T：

正T（先买后卖）: 早盘低位买入 → 拉高后卖出等量底仓 → 赚取价差
反T（先卖后买）: 早盘高位卖出底仓 → 回落后买回等量 → 赚取价差
```

**关键约束**：
- 当日做T股数 ≤ 底仓数（不能超卖）
- 单日做T上限 = 底仓的50%（留足安全边际，防止单边行情踏空）
- 收盘前必须平掉所有T仓位（T+1规则下当日买入部分次日才能卖，但用底仓替代则无此问题）

## 二、四种做T子策略

### Strategy A: 波动带做T (VolBand) — 保守型

**核心思想**：利用ATR（平均真实波幅）构建日内交易带，只在价格偏离开盘价超过阈值时触发。

| 参数 | 值 | 含义 |
|------|-----|------|
| `VOL_BAND_MULT` | 0.6 | 买入带 = 开盘价 - 0.6×ATR |
| `VOL_SELL_MULT` | 0.8 | 卖出带 = 开盘价 + 0.8×ATR |
| 网格 | 关闭 | 纯波段模式 |

**触发逻辑**：
```
IF 盘中最低价 ≤ 开盘价 - 0.6×ATR (买入触发)
   AND 盘中最高价 ≥ 买入价 + 0.8×ATR (卖出触发)
   THEN 正T: 低位买入 → 高位卖出 (赚取 0.8×ATR 价差)

IF 盘中最高价 ≥ 开盘价 + 0.9×ATR (卖出触发)
   AND 盘中最低价 ≤ 卖出价 - 0.6×ATR (买回触发)
   THEN 反T: 高位卖出 → 低位买回
```

**特点**：触发频率低（~9.5%交易日），但单笔收益大（日均10万元/笔）。

### Strategy B: 网格做T (Grid) — 高频型

**核心思想**：在开盘价上下设置多层等差网格，价格每触及一层就执行一次低买高卖。

| 参数 | 值 | 含义 |
|------|-----|------|
| `GRID_LEVELS` | 4 | 4层网格 |
| `GRID_STEP_PCT` | 1.2% | 每层间距1.2% |
| `VOL_BAND_MULT` | 1.5 | 宽波动带（基本不触发波段模式） |

**网格分布示例**（股价500元时）：
```
卖4: 524.0 (+4.8%)    网格4: +4.8%
卖3: 518.0 (+3.6%)    网格3: +3.6%
卖2: 512.0 (+2.4%)    网格2: +2.4%
卖1: 506.0 (+1.2%)    网格1: +1.2%
──────────────────    开盘价: 500
买1: 494.0 (-1.2%)    网格1: -1.2%
买2: 488.0 (-2.4%)    网格2: -2.4%
买3: 482.0 (-3.6%)    网格3: -3.6%
买4: 476.0 (-4.8%)    网格4: -4.8%
```

**特点**：触发频率高（~31.5%交易日），单笔收益小但积少成多，适合震荡行情。

### Strategy C: 动量做T (Momentum) — 进攻型

**核心思想**：牛市只做正T（先买后卖），熊市只做反T（先卖后买），顺势而为。

| 参数 | 值 | 含义 |
|------|-----|------|
| `VOL_BAND_MULT` | 0.4 | 窄买入带（积极入场） |
| `VOL_SELL_MULT` | 0.6 | 窄卖出带（快速止盈） |
| 网格 | 1层/2% | 辅助网格 |

**趋势判断**：
```
牛市 (trend_bull): 收盘 > MA20 AND MA5 > MA20 → 只做正T
熊市 (trend_bear): 收盘 < MA20 AND MA5 < MA20 → 只做反T
震荡 (sideways):   → 正T + 反T都可
```

**特点**：在长飞光纤的强势上涨周期中表现最佳（年化106.58%），但需警惕趋势反转时的回撤。

### Strategy D: 自适应复合 (Adaptive) — 推荐实盘

**核心思想**：综合上述三种策略，根据市场状态动态调整参数。

| 市场状态 | 策略倾向 | 参数调整 |
|----------|---------|---------|
| 高波动（振幅>7%） | 波动带为主 | 收窄波段参数，快速止盈 |
| 低波动（振幅<3%） | 网格为主 | 缩小网格间距 |
| 放量突破（量比>2） | 动量跟随 | 追涨不追跌 |
| 缩量盘整（量比<0.6） | 不做T | 全平等待 |

**特点**：平衡收益与风险，胜率稳定，适合长期运行。

### 5.2 实盘优化建议

#### A. 盘中tick触发（关键改进）

回测用的是日线OHLC模拟，实盘中建议改进为盘中tick触发：

```python
# 伪代码：实盘tick级触发
def on_tick(ContextInfo):
    tick = ContextInfo.get_full_tick([code])
    last_price = tick[code]['lastPrice']

    if last_price <= buy_target and today_buy == 0:
        # 触发T买入
        order_shares(code, t_shares, 'LIMIT', buy_target, ContextInfo, acc_id)

    if last_price >= sell_target and today_buy > 0:
        # 触发T卖出（平仓）
        order_shares(code, -today_buy, 'LIMIT', sell_target, ContextInfo, acc_id)
```

#### B. 盘口流动性检查

```python
# 避免在流动性不足时做T
tick = ContextInfo.get_full_tick([code])
bid_vol = tick[code].get('bid_vol', [0]*5)  # 买1-买5量
ask_vol = tick[code].get('ask_vol', [0]*5)  # 卖1-卖5量

if sum(bid_vol[:3]) < 50_000_000:  # 前3档买盘 < 5000万
    return  # 流动性不足，暂停做T
```

#### C. 时间窗口控制

```python
import datetime
now = datetime.datetime.now()

# 只在前2小时和后1小时做T（流动性最好）
morning_start = now.replace(hour=9, minute=35)
morning_end = now.replace(hour=11, minute=0)
afternoon_start = now.replace(hour=14, minute=0)
afternoon_end = now.replace(hour=14, minute=50)

if not ((morning_start <= now <= morning_end) or
        (afternoon_start <= now <= afternoon_end)):
    return  # 不在最佳时段，观望
```

### 5.3 风控机制

| 风控规则 | 触发条件 | 动作 |
|----------|---------|------|
| **单笔止损** | 单笔T亏损>3% | 立即平仓，当日不再做T |
| **日止损** | 当日累计亏损>1%总资金 | 平所有T仓位，当日停止 |
| **连亏熔断** | 连续3日做T亏损 | 暂停做T 2个交易日 |
| **缩量过滤** | 当日量比<0.6 | 全天不做T |
| **极端RSI** | RSI>85或<15 | 单边行情不做反向T |
| **重大消息** | 公司公告/行业政策 | 消息日全天不做T |

## 六、回测局限性

### 6.1 ⚠️ 日线OHLC模拟的天生缺陷

| 假设 | 回测 | 实盘 | 影响 |
|------|------|------|------|
| 买入执行 | 最低价触及=成交 | 需要对手盘卖给你 | **高估成交率20-40%** |
| 卖出执行 | 最高价触及=成交 | 需要对手盘买你的 | **高估成交率20-40%** |
| 流动性 | 无限流动性 | 盘口深度有限 | **大单冲击成本** |
| 同时成交 | 买卖同时完成 | 买卖有时间差 | **存在踏空风险** |
| 滑点 | 固定0.1% | 波动大时更大 | **实际成本更高** |

> **经验修正系数**：对于长飞光纤这类高流动性标的，实盘收益约为回测的 **35-50%**。即策略D的年化做T收益约在 **31-44%** 区间（而非回测的88.9%）。

### 6.2 回测周期特殊性

回测覆盖了长飞光纤的"超级周期"（+2052%），这种极端涨幅不可线性外推：

- **有利因素**：强趋势使得动量策略完美运行
- **不利因素**：未来如果进入盘整或下跌，策略收益将大幅缩水
- **合理预期**：在正常行情下（年波动30-50%），做T年化增强收益约 **15-30%**

### 6.3 未考虑的因素

- **印花税**：T+0交易的双向印花税成本（实盘中卖出收取0.1%）
- **冲击成本**：大额做T对盘口的冲击（尤其在价格快速变动时）
- **交易限制**：单账户日交易次数/金额上限
- **停牌风险**：持仓股票停牌期间无法做T

## 八、总结与建议

### 核心结论

1. **长飞光纤是做T的优质标的**：高价+高波动+强趋势+好流动性，四要素齐备
2. **实盘推荐策略D (自适应复合)**：平衡收益与风控，在不同行情下自适应调整参数
3. **合理预期年化做T增强收益**：**20-35%**（已做50%回测折价修正）
4. **必须盘中tick触发**：日线回测只是策略逻辑验证，实盘需要tick级执行
5. **严格风控不可省略**：缩量不做T、RSI极端不追、连亏熔断

### 实盘启动建议

| 阶段 | 时长 | 操作 |
|------|------|------|
| **模拟盘验证** | 2-4周 | 在QMT模拟盘中运行，验证成交率和滑点 |
| **小仓位试跑** | 1-2月 | 用10-20%底仓做T，积累实盘数据 |
| **逐步放大** | 2-3月 | 根据实盘胜率和收益，逐步提升至30-50%底仓 |
| **全量运行** | 持续 | 稳定运行，定期复盘优化参数 |
