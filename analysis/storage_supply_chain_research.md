## 一、市场叙事 → 真实的系统变化

### 当前市场在讲什么故事？

AI 算力竞赛已经进入"Agent 时代"——有状态的、长时间运行的 AI 智能体（Agentic AI）正在吞噬海量内存。市场从 2024-2025 年聚焦 GPU 算力卡，逐步转向认识到**存储才是更长、更难解决的物理瓶颈**。

### 什么系统变化在驱动需求？

| 变化维度 | 旧范式 | 新范式 | 物理约束 |
|----------|--------|--------|----------|
| AI 工作负载 | 单次推理（stateless） | Agent 长程任务（stateful, KV cache, 上下文窗口） | 单 session 内存需求从 GB 级跃升至 TB 级 |
| 上下文窗口 | 32K tokens | 256K → 1M+ tokens | 内存消耗指数增长 |
| CPU:GPU 比 | 1:4 | 向 1:1 靠近 | CPU 侧 DRAM 需求从 4-8GB/核 → 16-32GB/核 |
| 存储介质 | DRAM 为主 | HBM（近存计算）+ 企业级 SSD + HBF (High Bandwidth Flash) | HBM 消耗 **3x 标准 DRAM 晶圆** |
| 服务器存储 | 单机 64TB eSSD (2023) | 单机 256TB eSSD (2026) | NAND 需求 4 年翻 4 倍 |
| Token 需求 | 2025 基线 | Jensen Huang 预测 2030 年增长 **40x** | 存储需求跟随线性 → 超线性增长 |

### 核心判断

**存储已经从 AI 基础设施的"配套环节"升级为第一稀缺资源。** 市场过去 18 个月过度关注 GPU 算力，现在正在重新定价存储——但大部分人仍未充分认识到：这不是一次普通的周期性涨价，而是一场**以年为单位的结构性供给危机**。

## 三、稀缺层判定 — 哪里是真正的供需瓶颈？

用 Serenity 框架的判断标准来排：**供应商数量、扩产周期、技术壁垒、客户认证难度、产能锁定程度**。

### 第一稀缺层 (Hard Bottleneck): HBM / 先进 DRAM 晶圆制造

| 维度 | 判断 |
|------|------|
| **供应商** | 仅 3 家（三星 / SK 海力士 / 美光），中国长鑫尚未进入 HBM |
| **扩产周期** | 新 Fab 2-3 年 + 良率爬坡 → 最早 2028-2029 才有实质增量 |
| **产能锁定** | 2026 年 HBM 产能 100% 售罄，OpenAI 锁定 ~40% 至 2029 |
| **供需缺口** | SEMI China 估算 HBM 缺口 50-60% |
| **技术壁垒** | HBM4 需要 2048-bit 接口，HBM4E 需要 12 层以上堆叠 |
| **物理约束** | HBM 消耗 3x 标准 DRAM 的晶圆产能；DRAM 密度年增速从 ~45% 降至 ~9% |
| **价格信号** | 2026 Q2 DRAM 合约价环比 +57~64%，HBM 合同价 +20% |

> **这是整个 AI 存储链条的"终极卡点"。** 黄仁勋 2026 年 6 月在首尔公开表态："这不是周期性短缺，而是多年结构性行业危机。"

### 第二稀缺层 (Physical + Qualification Bottleneck): 先进封装 (CoWoS + HBM 堆叠)

| 维度 | 判断 |
|------|------|
| **供应商** | CoWoS 几乎 100% 台积电；HBM 堆叠封装三星/SK/台积电/Amkor |
| **扩产周期** | 台积电 2026 年 CoWoS 月产能 12 万片（较 2024 年底 +4x）但仍超额认购 |
| **产能锁定** | NVIDIA 锁定 ~60%，AMD ~11%，Google/Amazon ~15% |
| **物理瓶颈** | 2026 年核心矛盾已从"产能数量"变为"大尺寸封装的翘曲控制+热管理" |
| **技术壁垒** | 5.5x 光罩尺寸 → 9.5x 光罩 (2027)，110×110mm² 封装中 CTE 失配导致良率风险 |
| **设备依赖** | TCB 热压键合设备 2026-2028 年 HBM 端单独新增 532/648/748 台 |
| **持续时间** | 供需失衡至少延续至 2027 年中 |

> **封装已经从"后端配套"变成了"决定 AI 算力上限的核心变量"。** HBM4E 之争的胜负将取决于谁能通过台积电封装线的系统级验证。

### 第三稀缺层 (Tooling Bottleneck): 半导体制造设备

| 维度 | 判断 |
|------|------|
| **EUV 光刻机** | ASML 全球垄断，供应严格受限。存储厂与逻辑厂争抢同一设备池 |
| **刻蚀/沉积设备** | Lam/TEL/AMAT 三家垄断 80%+，交期 6-18 月 |
| **无尘室** | 摩根士丹利指出这是"第一物理瓶颈"，建设周期 3-4 年 |
| **检测设备** | 3D 堆叠（HBM）需要全新检测方案，产能紧张 |
| **国产突破** | 北方华创在长鑫设备采购占比高达 45%；中微刻蚀机 300+ 台量产 |
| **设备订单信号** | 长鑫 IPO 募资 295 亿投向产能升级 → 设备采购 174 亿，2026-2027 分批搬入 |

> **扩产的扩产才最确定。** 设备是吃"存储厂资本开支"最直接的环节。全球 WFE (Wafer Fab Equipment) 2026 年预计超 1300 亿美元。

### 第四稀缺层 (Structural Shortage from Cannibalization): NOR Flash / SLC NAND

| 维度 | 判断 |
|------|------|
| **核心逻辑** | 大厂将成熟制程产能转去做 HBM → 挤出 NOR Flash/SLC NAND 供给 |
| **价格信号** | 2026 H1 NOR Flash 合约价累计 +100~120%，SLC NAND +130~150% |
| **供应商缩减** | 海外大厂逐步退出小容量 SLC NAND |
| **需求端** | 汽车 ADAS + 边缘 AI + 工业 IoT 三股力量同时拉高容量需求 |
| **车规级** | TrendForce 称车规 SLC NAND 是整个存储市场"最强势"的细分 |
| **持续性** | 只要 HBM 继续吸走产能，这个挤出效应就不会消失 |

> **这是"AI 存储繁荣的副产品"——一个被主流叙事忽略的高弹性细分。** 高容量 NOR Flash (≥256Mb) 和车规 SLC NAND 下半年预计再涨 60-75%。

### 第五稀缺层 (Derivative Bottleneck): 存储主控芯片 + DDR5 接口芯片

| 维度 | 判断 |
|------|------|
| **SSD 主控** | 慧荣 (SIMO, ~30% 份额) + 群联 + 联芸科技三强格局 |
| **DDR5 接口** | 澜起科技全球 >40%，DDR5 RCD/MDB 芯片 |
| **需求推手** | AI 数据中心推动 PCIe 5.0/6.0 和企业级 SSD 主控需求 |
| **紧俏程度** | PCIe 5.0/6.0 高端主控可能开始紧张（非当前核心瓶颈） |
| **利润质量** | 主控/接口芯片是 Fabless 模式，毛利率高（澜起 58%），无晶圆厂重资产风险 |

### 为什么存储模组排不进前五稀缺层？

存储模组（江波龙/佰维/德明利）是**涨价弹性的受益者**，不是供应链的卡点。它们赚的是：

1. 低位囤库存 → 高价卖出 → 库存重估收益（2026Q1 江波龙净利暴增 2644% 就是这个逻辑）
2. 涨价周期下的营收扩张

但它们**不控制任何稀缺环节**——NAND 颗粒从原厂买、DRAM 颗粒从原厂买，封测外包或自建（门槛远低于前道制造）。当原厂产能紧张时，模组厂反而拿不到货。**在供给过剩期它们受益（低价囤货），在供给紧张期它们也难受（拿不到货）。**

> **排层级时的关键判断：能控制稀缺层 > 能供应稀缺层 > 能受益于趋势但无议价权。** 存储模组属于第三类。

## 五、公司优先研究清单

> 排序逻辑：**离稀缺层越近、证据越强、市场可能尚未充分定价的因素权重越高。** 这不是买入建议，是"最值得花时间深入研究"的排序。

### 🔴 Tier 1 — 最接近核心瓶颈，必须研究

#### 1. 美光科技 (MU, 美股) — HBM/DRAM 三大原厂之一，最直接卡位

| 维度 | 评估 |
|------|------|
| **卡住的环节** | HBM + 先进 DRAM 晶圆制造（第一稀缺层核心玩家） |
| **产业链位置** | 全球三大 DRAM IDM 之一（市占 ~20%），HBM4/HBM4E 量产中 |
| **排序原因** | 只有三家能做 HBM，它是其中之一。2026 年产能全部售罄。新品 HBM4E 正在 ramp |
| **证据强度** | 强。2026Q3 财报营收 414.56 亿美元（+345% YoY），毛利率 84.9%，Q4 指引 490-510 亿 |
| **机构目标价** | TD Cowen $1500 / Stifel $1500 / Wedbush $1300 / 大摩 $1050 |
| **市场可能没看清的** | 市场对 HBM 需求持续性的定价可能还不足。Agentic AI 带来的 DRAM 消耗是指数级的 |
| **主要风险** | 1) HBM 产能扩张慢于预期  2) 地缘政治限制对华出货  3) 当前估值已在定价大量好消息 |
| **下一步验证** | 跟踪 HBM4E 良率爬坡进度；关注公司对 2027 年产能的指引 |

#### 2. SK 海力士 (000660.KS / 正在推进美股 ADR) — HBM 市占第一

| 维度 | 评估 |
|------|------|
| **卡住的环节** | HBM 晶圆制造（目前 HBM 市占最高，是 NVIDIA HBM3/HBM3E 主供） |
| **产业链位置** | DRAM 全球 #2 (~35%)，HBM 领跑，HBM4E 16Gbps/pin 样品已出货 |
| **排序原因** | HBM 领先者，NVIDIA 核心供应商。五年晶圆产能翻倍计划验证了长期紧缺判断 |
| **证据强度** | 强。公开宣布 2026 年 HBM 产能已全部售罄；判断供应缺口可能延续至 2030 年 |
| **主要风险** | 1) 三星在 HBM4E 上追赶  2) 韩国政治/地缘风险  3) ADR 上市时间不确定 |
| **A股投资路径** | 华泰柏瑞中韩半导体 ETF（唯一可投的公募基金，但须注意 QDII 溢价）；太极实业（通过海太半导体为 SK 海力士提供 HBM 封装服务） |

#### 3. 台积电 (TSM/2330.TW) — CoWoS 先进封装唯一主导者

| 维度 | 评估 |
|------|------|
| **卡住的环节** | CoWoS 先进封装（第二稀缺层的绝对控制者） |

## 七、什么情况说明这个判断错了？

1. **AI CAPEX 大幅削减**（概率：低~中）
   - 如果 9 大云厂商削减 2026-2027 年 AI 基础设施投资，存储需求预期将崩塌
   - 跟踪信号：微软/谷歌/亚马逊/Meta 的季度 CAPEX 指引

2. **消费端需求破坏远超预期**（概率：中）
   - 手机出货量预测已下调 16%，PC 也在走弱。如果消费电子崩盘，NAND 可能从短缺转为过剩
   - 跟踪信号：全球手机/PC 季度出货量，终端产品零售价格

3. **中国长鑫/长存产能超预期释放**（概率：中~高）
   - 长鑫 2026Q1 营收 508 亿，如果 IPO 后加速扩产，可能在中低端 DRAM 市场形成价格压力
   - 但 HBM 不受影响（长鑫尚未进入），所以高端和低端可能出现分化
   - 跟踪信号：长鑫月产能数据、价格变化、美国对华设备管制动态

4. **替代技术路线突破**（概率：低）
   - HBF (High Bandwidth Flash) 标准化推进 —— Sandisk+SK 海力士推动，NAND 堆叠成 HBM-like 封装
   - 如果 HBF 能部分替代 HBM，HBM 的稀缺性溢价可能下降
   - 跟踪信号：HBF 标准进度，2026H2 首批样品，2027 年初首批设备

5. **地缘政治极端情景**（概率：低，但影响极大）
   - 台海危机 → 台积电 CoWoS 断供 → 全球 AI 供应链断裂
   - 更严厉对华芯片制裁 → 长鑫/长存设备断供 → 国产存储泡沫破裂
   - 跟踪信号：美国商务部 BIS 规则更新、台海局势、ASML 对华出货情况

6. **估值已经把好消息定价了**（概率：中~高）
   - 美光年内涨幅已超 265%，A 股存储概念股总市值超 8.2 万亿
   - 长江存储/长鑫 IPO 可能成为阶段性"利好出尽"
   - 跟踪信号：板块估值分位数、IPO 定价 vs 二级市场反应

## 九、总结 — 可以怎么用这份研究？

### 如果只记三件事

1. **存储缺口的本质是物理+时间问题，不是周期性波动。** HBM 供需缺口 50-60%，新晶圆厂 3-4 年才能产出，DRAM 密度提升已从每年 45% 降到 9%。这轮紧缺至少持续到 2028 年。

2. **最有价值的环节在上游：设备 > 制造 > 封装 > 设计 > 模组。** 全球设备和先进封装是最接近真实物理约束的环节。A 股里，设备（北方华创/中微/精智达）和接口芯片（澜起科技）确定性最高。模组弹性大但不控制稀缺层。

3. **两条并行主线：全球 HBM 紧缺（投美光/SK/设备）+ 中国存储国产化（投长鑫/长存/长江存储链设备商）。** 它们逻辑不同但有共振——全球缺货推高价格，国产替代加速推进。

### 当前研究结论的置信度

| 判断 | 置信度 | 说明 |
|------|--------|------|
| HBM 是存储产业链最稀缺的环节 | 高 | 多源交叉验证（SEMI、黄仁勋、美光、SK、大摩） |
| 先进封装 (CoWoS) 是第二瓶颈 | 高 | 台积电产能数据 + 物理约束分析 |
| 全球存储设备交期 6-18 个月 | 高 | 多份产业报告验证 |
| 本轮周期至少持续至 2028 年 | 中高 | 产业界一致预期，但取决于 AI CAPEX 持续性 |
| NOR Flash 涨价弹性被低估 | 中 | TrendForce 数据强，但市值小、流动性限制 |
| 消费端需求破坏的程度 | 低 | Jefferies 和 Bernstein 分歧极大 |
| 中国存储厂商 (长鑫/长存) 的远期竞争力 | 中低 | 依赖制裁环境和技术迭代速度，太多未知变量 |
